## §2 第一性原理理论结构

**功能**：建立框架的数学基础。从 Transformer 最底层的数学对象——残差流张量积——出发，区分语义空间和路由空间两个正交因子，分析各自的范畴状态和因果耦合，由此导出收敛二分、截断投影、记忆二分，然后推进到边缘混沌的动力学图景，最后在贝叶斯类比中诚实标注剩余的结构不匹配。
**论证强度**：严格层（2.1.1-2.1.3, 2.2）+ 形式论证层（2.3, 2.4）+ 概念构造层（2.5）+ 类比层（2.6）。

### 2.1 残差流的数学结构：ℝⁿ ⊗ ℝᵈ 分解

Transformer 每一层的状态是一个矩阵：

```
X^(l) ∈ ℝⁿ ⊗ ℝᵈ

n = token 数量（序列长度）
d = 特征维度

X^(l) = [x₁^(l), x₂^(l), ..., x_n^(l)]ᵀ
```

第 l 行 x_i^(l) ∈ ℝᵈ 是 token i 在当前层的语义向量。同一个矩阵 X^(l) 里装着两样东西：**语义**（每行的向量 ∈ ℝᵈ）和**通路**（行与行之间的注意力权重 ∈ ℝⁿ 上的矩阵）。它们是同一个张量积的两个正交因子——各自独立运算，不能互相归约，但因果耦合。

张量积结构意味着 ℝⁿ 和 ℝᵈ 两个因子的运算是**独立的**——A 在行方向混合 token（ℝⁿ 上的运算），W 在列方向变换特征（ℝᵈ 上的运算），先做哪个后做哪个互不干涉。**但正交 ≠ 无关**——A 的条目依赖 x ∈ ℝᵈ 的值，W 改变 x，所以 W 的输出 → A 的输入变化 → 下一层 A 不同。两者构成反馈环。

以下 2.1.1-2.1.3 分别考察两个因子的范畴状态及它们的因果耦合。

#### 2.1.1 语义空间 ℝᵈ：范畴闭合

ℝᵈ 上的操作由一组固定的线性算子——权重矩阵 W_Q, W_K, W_V, W_O 和 MLP 矩阵——完成。它们不随输入变化，是训练后固定的参数。

**ℝᵈ 构成范畴（Mat / Vect）**：
- 对象：ℝᵈ 及其子空间
- 态射：W 矩阵——从 ℝᵈ 到 ℝᵈ 的线性映射
- 复合：W₂ · W₁ 始终定义良好（矩阵乘法），结合律由矩阵乘法保证
- 恒等：单位矩阵 I_d

**米田引理在 ℝᵈ 上成立**：语义空间 ℝᵈ 的身份完全由所有指向它的态射 hom(-, ℝᵈ) 确定——即所有 W 矩阵及其复合的全体。ℝᵈ 有自足的、不依赖 token 地址那一端的代数结构。

**单层注意力是合法的 FinStoch 态射**（来源：闭合分析缺口一、缺口二第一步，均已严格闭合）：
- 单头注意力 A = softmax(QKᵀ / √d_k) 严格产出随机矩阵（行和为 1，非负）
- 多头注意力：多个随机矩阵经 concat + W_O 做凸组合，随机矩阵集合是凸集，结果仍为随机矩阵
- 因此：单层注意力 A 是 FinStoch 中的合法态射——这一步**没有近似，没有比喻**
- 米田嵌入的操作化：一个 token 的语义身份由所有其他 token 到它的条件概率关系所定义（分布语义学"you shall know a word by the company it keeps"的范畴论精确表述）

**训练将全部语义知识压进了 ℝᵈ 的几何中**——向量的方向、距离、W 矩阵的变换。W 矩阵的范畴复合不是"模拟"语义变换——它就是语义变换。Anthropic 的虚拟权重和路径展开，做的就是把 W 的复合作为语义操作来读。

**论证强度**：严格层。W 矩阵构成 Mat/Vect 态射是数学事实；单层 softmax 注意力产出随机矩阵是数学事实；米田嵌入在有限维向量空间上成立是范畴论标准结论。

#### 2.1.2 路由空间 ℝⁿ：跨层范畴不闭合

注意力矩阵 A ∈ ℝⁿ → ℝⁿ，条目 a_{ij} = softmax(Q(x_i) · K(x_j)ᵀ / √d) 表示"token i 应该从 token j 取多少信息"。这是 token 间的路由决策。

**单层内**：对固定的表示 X^(l)，A^(l) 是 FinStoch 态射（行随机矩阵）。成立。

**跨层**：A^(l+1) ∘ A^(l) **无定义**。这是全框架最关键的结构事实之一。原因：

```
a_{ij}^(l) = f(x_i^(l), x_j^(l); W_Q, W_K)   ← 依赖 ℝᵈ 当前状态
x^(l+1) = x^(l) + A^(l)·V^(l)·W_O + MLP(...) ← ℝᵈ 状态被改变
→ A^(l+1) 的输入在 ℝᵈ 端已被 A^(l) 的输出改变
→ A^(l+1) 的域对象不是 A^(l) 的余域对象
→ Chapman-Kolmogorov 方程不成立
→ 复合 A^(l+1) · A^(l) 不是适当的数学描述
```

ℝⁿ 不构成跨层范畴的结构原因：
- a_{ij} 不由地址 {i, j} 决定——它由地址里装的**语义内容**决定
- 语义内容每层被 W 改变 → a_{ij} 逐层变化
- a_{jk} 和 a_{ij} 之间没有运算能得到 a_{ik}——不传递
- 没有恒等态射——自注意力 a_{ii} 做计算，不是"什么都不做"

**米田引理不适用**：token 位置不构成范畴对象——米田上场的前提不满足。Token i 的身份不能由"指向它的注意力"确定，因为注意力本身依赖 ℝᵈ 端当前状态。Token 是空槽位——索引 i 跨层不变（"哪里"），但里面装的向量 x_i^(l) 每层被重写（"什么"）。Token 没有自足的语义身份——它的身份完全寄生在语义空间 ℝᵈ 上。

**反事实推理：如果 ℝⁿ 也构成范畴会怎样**（这是关键检验）：

```
假设：米田成立，a_{ij} 由地址 {i, j} 自足决定，不依赖 ℝᵈ
→ A 跨层固定 → A_eff = A^L
→ {1,...,n} 上的 L 步 Markov 链，稠密正矩阵
→ 收敛到唯一平稳分布 π
→ A_eff 所有行 = π（所有 token 注意力分布相同）
→ 所有 token 收到相同的加权上下文 → 同方向更新 → 表示同质化
→ 跨 token 动态路由消失 → 语义结构丰富性严重退化
→ 深度冗余（注意力层不再提供有效信息增量）
```

**复合闭合 = 跨 token 动态路由消失 = 语义死亡。** 这不是设计疏忽——ℝⁿ 端的不复合是语义聚类能够在 ℝᵈ 端形成的数学边界条件。

**论证强度**：严格层（跨层复合失败是数学事实）+ 概念构造层（"不复合 = 功能性"的论证）。

#### 2.1.3 两个因子的因果耦合

两个因子的范畴状态不是独立的——它们因果耦合：

```
W 矩阵构成范畴 → W 复合改变语义空间坐标 → 每一层面对的 ℝᵈ 状态不同
  → A 矩阵依赖 ℝᵈ 当前状态（Q·Kᵀ 使用当前坐标）
  → 语义空间变了 → A 矩阵必然逐层变化
  → A 的跨层复合断裂
```

**W 的复合成功是 A 的复合失败的原因。** 这不是两个独立事实，是因果关系。只要语义空间在演化（W 复合），路由空间就不能累积（A 不复合）。

**不对称闭合**：
- ℝᵈ 端：范畴闭合，复合成立，米田成立 → 语义有自足结构
- ℝⁿ 端：范畴不闭合，复合失败，米田不适用 → 路由寄生在语义上
- 因果方向：ℝᵈ 的闭合 → ℝⁿ 的不闭合

这个不对称不是需要修复的结构缺陷——它是 Transformer 能够逐层演化语义空间、产生复杂语义聚类的数学前提。如果两端都闭合（A 可复合）→ 语义死亡。如果两端都不闭合（W 不可复合）→ 无语义自足结构 → 深度坍缩为单层。

**论证强度**：严格层（因果依赖是数学事实）+ 形式论证层（不对称闭合作为功能条件的推导）。

### 2.2 收敛的操作化定义

- 三条判据：损失稳定、参数更新趋零、行为可预测
- "收敛约束"的精确含义：经历了梯度反馈锻造的信息享有更高可靠性
- 核心区分——**收敛二分**：不是所有信息都经过了收敛——参数经过了，上下文没有
  - 参数：训练中经梯度下降在训练分布上达到稳定配置 → 享有收敛约束
  - 上下文：每次推理时动态注入，不经梯度锻造 → 不受收敛约束
- 这个区分的结构根源现在可追溯至 §2.1：参数 = ℝᵈ 端的 W 矩阵（固定，范畴闭合）；上下文 = ℝⁿ 端的路由实例（每次动态参数化，跨层不复合）
- 来源：收敛表述_定义段

**论证强度**：严格层（收敛的操作化定义是约定性的；参数/上下文的区分对应 §2.1 的数学结构）。

### 2.3 截断投影

- 参数空间有效维度远小于名义维度（Li 2018, Aghajanyan 2020, LoRA, Xu 2026）
- 模型经训练收敛后得到的稳定配置是米田嵌入在低维参数流形上的截断投影（"学到映射"指收敛稳定后的配置，其函子性质是经验发现而非定义保证——详见§6.1.3）
- 核心约束：超出截断投影覆盖范围的语义态射不受收敛保证
- **直接后果**：基座模型直接对话时，输入易超出截断投影覆盖范围，导致分布外暴露和续写模式泄露——泄露概率高是截断投影的结构性后果，非偶然事故（草稿第8条）。人类语言本身也不具有天然的逻辑结构，仅是对逻辑的模拟——两者在"语言层面的逻辑态射未经严格收敛"这一点上同构
- **与 §2.1 的衔接**：截断投影描述的是 ℝᵈ 端语义空间的容量限制；ℝⁿ 端路由空间的不复合（§2.1.2）描述的是路由本身的不可累积性。两者正交且互补——语义空间有限（维度约束）+ 路由空间不复合（结构约束）→ 任何单次前向传播的信息处理能力受双重限制
- 来源：草稿5-8条，硬伤分析三

**论证强度**：严格层（参数空间内在维度研究是经验事实）+ 形式论证层（截断投影→覆盖边界→分布外暴露的推理链）。

### 2.4 记忆的结构性二分

- 长期记忆（参数空间中的点，空间结构）vs 短期记忆（上下文序列，顺序结构）
- 两种结构属于不同数学范畴 → 不存在连续过渡的"中期记忆"
  - 长期记忆：静态点配置，无时间维度，由 ℝᵈ 端的 W 矩阵承载
  - 短期记忆：序列，由 ℝⁿ 端路由在语义空间 ℝᵈ 中产生的轨线实现
- 位置编码提供顺序，不提供时间性——无衰减、无巩固、无自发重组
- 来源：草稿13-16条

**论证强度**：形式论证层（从数学范畴的不同结构推导"不存在连续过渡"）。

### 2.5 边缘混沌：不复合的动力学后果

§2.1.2 确立了注意力跨层不复合是数学事实。本节分析这个事实的动力学后果——不复合如何产生 Transformer 的复杂行为。

#### 2.5.1 不复合 → 语义聚类正反馈

```
相似 token → 相似 Q/K 向量 → 相似注意力模式 → 从上下文收取相似信息
  → 相似更新 → 保持相似 → 聚类自持（收敛面）

相异 token → 不同注意力模式 → 不同更新 → 更加相异 → 聚类分化（扩张面）
```

注意力不复合 = 路由每层重新决策 = 不被"平均化"约束 = 正反馈可以持续运作。如果注意力可复合（Markov 链）→ 长程收敛到平稳分布 → 所有 token 的注意力行相同 → 正反馈被平均化消灭。**不复合 = 聚类可以发生的条件。**

#### 2.5.2 混沌通路的三个条件

注意力不复合提供了**路径依赖**（第一步往哪走影响第二步往哪走——路由不由固定转移概率决定，而由当前语义状态决定）。但路径依赖单独不足以产生混沌——还需要两个条件，而 Transformer 恰好都满足：

| 条件 | 数学机制 | Transformer 中的实现 |
|------|---------|-------------------|
| 路径依赖 | 下一步依赖当前状态 | ℝⁿ 不米田 → a_{ij} 依赖 ℝᵈ 当前状态 |
| 非线性折叠 | 微小差异被放大为大位移 | Softmax 将 Q·K 小差异放大为概率单纯形大位移；ReLU/GeLU 阈值开关；LayerNorm 约束模长到球面 |
| 局部扩张 | Jacobian 在某方向特征值 > 1 | 残差 Jacobian J = I + ∂F/∂x，在某些方向上特征值 > 1，相邻轨线指数分离 |

三者同时成立 → **混沌通路**：微小的初始语义差异在通路迭代中被放大，相似起点的 token 在短期共向（收敛面），长期分叉（扩张面）。

**不米田是混沌的必要条件，不是充分条件**——如果 F 是线性的，路径依赖的系统只会收敛或发散，不会混沌。Transformer 之所以能维持复杂语义结构，是因为不米田 + 非线性 + 局部扩张共同产生的**边缘混沌**。

#### 2.5.3 边缘混沌假说

将残差流视为离散动力系统：

```
x^(l+1) = x^(l) + F(x^(l); W)

Jacobian: J^(l) = I + ∂F/∂x|_{x=x^(l)}
Lyapunov 指数: λ_i = lim_{L→∞} (1/L) Σ_{l=1}^L ln σ_i(J^(l))
```

| 动力学状态 | Lyapunov 谱 | 对 Transformer 意味着什么 |
|---|---|---|
| λ_max ≪ 0 | 过阻尼 | 轨线收敛到不动点 → 深度坍缩 → 所有 token 最终同质 |
| λ_max ≈ 0 | 边缘混沌 | 轨线既不收敛也不发散 → 深度有持续信息增量 → 语义结构可维持 |
| λ_max ≫ 0 | 混沌 | 轨线指数发散 → 表示爆炸 → 初值极端敏感 → 不可训练 |

**训练 = 在参数空间中寻找边缘混沌态。** 训练好的 Transformer 必须恰好坐在 λ_max ≈ 0 的临界线上。

这解释了已知但未系统解释的现象：
- **Over-depth 退化**：层数超过 Lyapunov 时间后，轨线在混沌吸引子上随机游走，新层不增加信息
- **残差不能去掉**：去掉恒等项 I → Jacobian 失去对角优势 → 谱失控 → 无法维持在边缘混沌
- **LayerNorm 的位置敏感性**：LayerNorm = 每步对状态做投影 → 约束轨线不跑出有界区域 → 混沌保持"有界非周期"
- **越深越难训**：长轨线对参数极度敏感 → 反向传播通过边缘混沌系统 → 梯度本身也边缘混沌

#### 2.5.4 残差加法是唯一能维持边缘混沌的累积方式

残差连接 `x^(l+1) = x^(l) + F(x^(l))` 是一个**离散 Euler 积分器**。为什么不用乘法？

```
若 x^(l+1) = x^(l) ⊙ F(x^(l))（乘法残差）：
  |F| < 1 → 轨线指数衰减到 0 → 深度坍缩
  |F| > 1 → 轨线指数发散 → 表示爆炸
  有界非线性 σ 下，几乎所有轨线趋向 0

加法残差 x^(l+1) = x^(l) + F(x^(l))：
  Jacobian = I + ∂F/∂x → 恒等项 I 维持当前位置记忆
  雅可比特征值可维持在 ≈1 附近 → 边缘混沌可持续
```

**加法是唯一能长期维持边缘混沌的累积方式 = 唯一能让深度持续产生复杂语义结构的积分器。** 乘法残差不能维持长期的复杂轨线。

**论证强度**：概念构造层。路径依赖和不复合的推导属于严格层；非线性折叠和局部扩张的识别属于形式论证层；Lyapunov 谱的分析是概念推导（尚未有直接的逐层 Jacobian 谱分解实验验证），但给出了可检验的预测（见 §5.2）。混沌三条件的"同时成立"判断来自对 Transformer 计算图的逐项分析而非数值验证，因此整体标注为概念构造层。

### 2.6 三层贝叶斯类比与态射复合的结构不匹配（诚实标注）

§2.1 的分析已经将"弱闭合"精确定位为 ℝⁿ⊗ℝᵈ 分解中的不对称范畴闭合。本节在此精确基础上重述三层贝叶斯类比，避免模糊的"弱闭合"标签。

**三层贝叶斯类比（四则真问题第2条）：**
1. 单头单层注意力 → 单次贝叶斯推断快照（以训练分布统计为隐式先验的核条件期望）——**类比成立**：单层 A 是严格的 Markov 核（§2.1.1）
2. 多层 Transformer 前馈 → 多轮信念精炼——**类比部分成立**：逐层更新表示类似变分推断的迭代近似。但关键不匹配在于：多轮贝叶斯更新中的"更新"是条件概率的 CK 复合（乘性），而 Transformer 的层间是残差加法 + ℝⁿ 路由不复合（§2.1.2-2.1.3）——层间不是纯粹的贝叶斯条件更新
3. 全程预训练 → 隐式贝叶斯模型平均——**类比部分成立**：权重空间上的近似边际化，但不是严格的贝叶斯推断

**第二层的关键结构不匹配（全框架最薄弱环节的精确定位）：**
- 贝叶斯类比要求：A_effective = A_L · ... · A_1（CK 积分 = 乘性复合）
- Transformer 实际：x_L = x_0 + Σ(A_l·V_l·W_l^O + MLP_l(...))（加性累积）
- 这不只是"加法 vs 乘法"——这是 ℝⁿ 端不构成跨层范畴（§2.1.2）在贝叶斯类比中的表现
- 策略：接受弱闭合——功能逼近，放弃严格对应。但现在的"弱闭合"有了精确的结构含义：ℝᵈ 闭合 + ℝⁿ 不闭合 + 因果耦合（不对称闭合），而非模糊的"有时成立有时不成立"
- 残差加法不仅仅是"与态射复合不匹配"——它是唯一能维持边缘混沌（§2.5）的积分器选择。加法不是贝叶斯复合的退化近似，它是 Transformer 动力学的功能基石
- §6.1.1 展开全貌

**论证强度**：类比层。三层对应的概念关系成立但第二层存在已精确刻画的结构不匹配。

**来源**：闭合分析缺口二第二步（仅弱闭合），但分析精度经 §2.1 大幅提升。

### 3.2 框架猜想陈述

**C1（收敛二分猜想——框架性主张）**：参数经过训练收敛 vs 上下文未经收敛——这一区分的所有可观察后果构成 Agent 行为差异的统一解释框架。此为框架性主张而非单一可检验猜想；其有效性由 C2-C6 的检验结果联合支撑。结构根源追溯至 §2.1 的不对称闭合。

**C2（截断投影猜想）**：Transformer 参数空间的有效维度限制使模型只能学到米田嵌入的低维截断投影。超出该投影覆盖范围的语义映射不受收敛约束。可检验推论：模型性能在训练分布边缘处出现系统性衰减，衰减速率与任务偏离训练分布的程度正相关。

**C3（根需求外在猜想）**：在参数冻结、截断投影、上下文不经收敛三项前提下，全自动闭环在根节点处不可闭合——最高层验证标准必须由系统外部提供。否证条件见 §5.4。

**C4（发散任务偏好依赖猜想）**：发散任务中，偏好信号的密度与方向一致性决定探索效率。稀疏偏好→停滞；矛盾偏好→漂移；分布外偏好→退化。可通过受控实验检验（T2, §5.2）。

**C5（自主性-可靠性互斥猜想）**：Agent 自主性增加等价于更多决策由上下文中的测试收敛完成，等价于可靠性下界降低。可检验部分：自主性程度与任务完成可靠性呈系统性反相关（T6, §5.2）。"不可消除"部分标注为框架性主张——任何技术进步均可被解释为"放宽"了而非"消除"了权衡，故该部分的否证条件需更精确的"消除"操作化定义；在精确定义完成前，框架主张"放宽"。

**C6（Agent Scaling 先行平台期猜想——待精确化）**：Agent 多步任务性能的 Scaling 曲线将系统性地早于单步预测进入平台期——因为前者上界由验证信号覆盖度决定，后者由参数空间有效维度决定。当前为方向性预测，精确化（平台期差距量级估计、拐点复杂度定位）见 §6.6。

**C7（AGI 不可达猜想）**：在当前 Transformer 训练范式下，LLM Agent 不可能实现人类意义上的 AGI。五项结构性缺失——语义空间不完备（截断投影）、模态不对齐（注意力限于语言）、动机不自发（根因未明，仅观察）、无渐进记忆（记忆结构二分）、顺序性而非时间性（位置编码）——各自有其独立论证链，汇聚于 §4.11。"AGI"的精确定义见 §3.1。

**C8（边缘混沌猜想——v4 新增）**：训练良好的 Transformer 处于边缘混沌态（λ_max ≈ 0）。过阻尼导致深度坍缩和表示同质化；混沌导致表示爆炸和不可训练。此猜想为 C1-C2 的动力学基础提供统一解释。可检验预测见 §5.2（T7-T10）。


## §4 证明或论证

**功能**：从 §2 的公理和 §3 的定义出发，形式化推导收敛任务的全部结构性结论。
**论证强度**：形式论证层（三段论形式）。

### 4.1 前提清单

| 编号 | 前提 | 来源 |
|:--:|------|:--:|
| P0 | 模型处理任何任务的"统筹"和"连贯"都以有限的文字输入为基础 | 草稿17 |
| P1 | 文字输入的有效边界划定了模型对单一问题的能力边界 | §2.3 → P1推导 |
| P2 | 超出边界的问题不受收敛约束覆盖 | §2.3 |
| P3 | Agent通过测试收敛逼近目标——可验证目标 + 反复测试 | §3.1 |
| P4 | 测试收敛依赖可验证目标 | §3.1 |
| P5 | 上下文映射未经收敛约束，不稳定 | §2.2（收敛二分）+ §2.1.2（ℝⁿ 跨层不复合 → 路由不受收敛保证） |
| P6 | Agent = 模型 + Harness | 草稿12 |

**§2.3→P1 的推导步骤**：输入文本通过注意力映射到参数空间 → 参数空间有效维度有限（§2.3）→ 模型只能可靠处理信息量不超出截断投影覆盖范围的输入 → 文字输入的有效边界由此划定。注意 P0（所有统筹依赖文字输入）是 P1 的元前提——不限于单一问题，而是声明模型赖以运作的唯一媒介就是文字输入。

**P5 的强化推导（v4 新增）**：上下文信息之所以"不经收敛"，结构根因在 §2.1.2——路由空间 ℝⁿ 跨层不构成范畴，每一层的注意力路由在当前语义状态下重新决策，不享有跨层的结构稳定性。这比 v3 中"上下文未经梯度锻造"的表述更深一层——它指明了未经收敛的**数学机制**。

### 4.2 桥接前提

| 编号 | 桥接 | 性质 |
|:--:|------|------|
| B1 | 多数有意义的任务超出单次文字输入边界 → 分解是必然的 | 经验观察 |
| B2 | 流程节点向下一次性传导 → 节点错误使下层全偏，代价乘性放大 | 结构推理 |
| B3 | 验证标准递归上溯终止于一个不可由系统自生成的权威 | 推导自 P1+P2+P4 |

**B3 推导**：验证链的每一层验证在各自的上下文窗口内进行（P1）→ 每层仅持有局部信息（P2）→ 最顶层需要全局信息做最终判断（"是否满足用户真实意图"）→ 该语义不在训练分布中（§2.3的截断投影边界）→ 验证链终止于一个不可由系统自生成的权威。

### 4.3 推导一：任务必须分解

多数有意义任务超边界（B1）+ 超边界信息不受收敛约束（P1+P2）→ 必须分解为落入边界内的子任务

### 4.4 推导二：子任务须可验证，分解形成层级

子任务目标须在边界内设定（P1）+ 需可验证目标触发测试收敛（P3+P4）+ 分解本身是一次测试收敛（P1应用到分解行为自身）→ 层级形成：上层收敛产出下层目标

### 4.5 推导三：层级节点须硬约束（三段论形式）

1. 上下文中的信息未经收敛（P5，其结构根因在 §2.1.2——ℝⁿ 跨层不复合）
2. 任务分解决策（节点如何切分）需要可靠的结构保持映射
3. 用未经收敛的信息做需要收敛保证的决策 → 结构不可靠

加上流程节点错误代价乘性放大（B2）→ 关键分叉点须由独立于模型的硬约束（人工定义或程序逻辑）介入

**工程验证**：长篇网文创作框架中，角色状态、时间线、伏笔库由独立程序逻辑追踪而非模型上下文记忆——程序硬约束锚定关键分叉点，模型不自判人物出场和伏笔回收时机。详见 §5.3 案例1。

### 4.6 推导四：根需求人工锚定

测试收敛依赖可验证目标（P4）+ 验证标准递归上溯（P4自反应用）+ 最上层验证标准不可由系统替代（B3）→ 递归终止于人工给定的最高层需求

### 4.7 综合：多级需求收敛结构

> 模型能力边界由文字输入边界划定。现实任务超边界→必须分解→层级形成→硬节点约束→根需求外在。
> 层级加深→收敛成本指数扩张。最高级目标须人工提出。全自动闭环在根节点处原则上不可闭合。

**"全自动闭环"论断的三层论证（四则真问题第3条）：**

| 层面 | 定义 | 不可闭合性质 | 论证强度 | 对应推导 |
|------|------|-------------|:--------:|:--------:|
| 目标设定层 | 系统能否自主生成最高目标 | 逻辑必然——根需求语义不在训练分布内 | 强 | §4.6推导四 |
| 任务分解层 | 系统能否在给定目标下自主规划子任务 | 结构不可靠——上下文驱动分解不受收敛保证（结构根因：ℝⁿ 跨层不复合） | 中 | §4.5推导三 |
| 执行闭环层 | 系统能否在给定验证标准下自主完成任务 | 条件可行——若验证信号覆盖充分，无需人工介入 | 弱 | §4.4推导二 |

框架的核心论证支持**目标设定层不可闭合**。"全自动闭环不可闭合"的精确范围限定于此层——不声称执行闭环层不可自动化。

### 4.8 从 Agent 缺失推导人的结构性角色

Agent 三项结构性缺失（§2.3 + §2.4 + 草稿26-31）：
1. 收敛映射维度有限（截断投影）
2. 记忆无时间性（记忆结构二分）
3. 模态缺失（注意力限于语言，感官不对齐）

→ 人必须补偿：

| 缺失 | 人的补偿角色 | 对应草稿 |
|------|-------------|:--:|
| 收敛不全 | 约束模型泄露；划分任务空间到上下文窗口内 | 32 |
| 记忆缺失 | 引入新动态知识（外部持久化+重新注入上下文） | 32 |
| 模态缺失 | 引入额外的生活体验式动机（偏好、审美、意图）——"生活体验式"限定词指人类认知的时间性和具身性 | 32 |
| 三者共同 | 锚定根需求（根需求语义在收敛空间外） | 33 |

**人的两种结构性角色**（草稿37-38）：
- **操作人**（收敛任务）：设定根需求、仲裁关键分叉、划分任务空间——人的上限即系统上限（原则51）
- **合作者**（发散任务）：持续注入偏好信号、匹配模型表现的理解与判断力——发散人机对话（原则48）

→ 人的结构性角色不由模型能力提升而消除——缺失来自架构数学结构（参数冻结、softmax 注意力、有限维截断、ℝⁿ 跨层不复合）

### 4.9 发散任务：偏好驱动迭代
**论证强度：概念构造层**

- 发散任务起始时 C_V = ∅（无可自动验证标准）
- 模型行为 = P(next_token | context) 自回归采样，无"主动探索"
- 方向性完全来自外部偏好信号对后续采样条件分布的连续重塑
- 人的偏好判断等价于收敛任务中损失函数的梯度——它提供方向
- 偏好信号的信息论约束：稀疏→停滞；矛盾→震荡；分布外→退化

### 5.2 框架的差异化可检验预测

#### 5.2.1 Agent 层预测（T1-T6，v3 既有）

**T1（Agent Scaling先行平台期）**：MMLU类单步评估的scaling曲线与Agent多步任务完成率的scaling曲线将出现系统性分歧——后者更早进入平台期。

**T2（发散任务偏好依赖）**：发散任务中，偏好信号的密度与方向一致性预测探索效率。稀疏偏好→探索停滞；矛盾偏好→产出漂移。可通过受控实验（固定候选生成器，操控偏好反馈的频率和一致性）检验。

**T3（发散→收敛过渡可靠性）**：过渡判断不由收敛保证。随任务复杂度增加，模型自主判断过渡时机的准确率将收敛至非零下界。可通过与人工过渡判断对比检验。

**T4（发散收敛混合Agent）**：独立上下文空间 + 显式过渡协议的Agent在混合任务上的完成率将高于共享上下文空间的Agent。可通过A/B实验检验。

**T5（逆缩放与验证稀疏度相关性）**：逆缩放现象在验证信号稀疏的任务域中更频繁出现。可按验证信号密度对任务域分层，检验各层中逆缩放的发生率。

**T6（自主性-可靠性反相关）**：Agent自主性程度与任务完成可靠性呈系统性反相关，且此反相关不由模型规模增加而削弱。可通过多模型-多自主性水平的交叉实验检验。

#### 5.2.2 架构底层预测（T7-T10，v4 新增，来源：笔记可检验预测）

以下四条预测测试的是框架的数学基础——不对称闭合和边缘混沌假说——而非框架的 Agent 层推论。它们更底层，也更危险（更容易被否证）：

**T7（语义聚类随层增强）**：已训练 Transformer 各层 token 表示的轮廓系数（silhouette score）应随层数保持高位或递增。预测机制：ℝⁿ 不复合 → 正反馈（相似趋同/相异趋分）→ 聚类随深度增强。若注意力跨层复合成立，轮廓系数应随层数衰减。

**T8（最大 Lyapunov 指数 ≈ 0）**：训练良好的 Transformer，逐层 Jacobian 的最大 Lyapunov 指数应在 0 附近——既不收敛（λ_max ≪ 0，坍缩）也不发散（λ_max ≫ 0，混沌）。检验方法：在标准 benchmark 模型上计算逐层 Jacobian 谱分解并估计 Lyapunov 指数。

**T9（残差强度调节分叉）**：将残差连接改为 `x + ε·F(x)`，当 ε → 0 时恢复深度坍缩（Jacobian → I，收敛到不动点，聚类消失）；当 ε → ∞ 时进入混沌（Lyapunov 指数 > 0，表示破缺）。可通过在已训练模型上插值 ε 并测量表示多样性来检验。

**T10（强制跨层复合 → 同质化）**：若通过架构修改强制注意力跨层复合（如注意力参数共享 + Chapman-Kolmogorov 损失），token 表示的聚类结构应随深度消失，模型表现应退化。这是最直接的否证实验——如果强制复合后模型表现不退化，则"不复合 = 功能性"的核心命题被否证。

### 5.3 自有工程验证案例

以下案例来自框架作者本人的工程实践，每一项都对应框架中多条原则的联合验证。案例按"框架预测→实现→结果"的格式展开。

#### 案例1：长篇网文多Agent创作框架

**问题**：长篇网络小说（>10万字）的AI辅助创作。多人物、多支线、跨章节伏笔。业界已有Agent在短篇和场景写作上可用，但长篇均出现主干叙事逻辑断点——模型在长上下文中丢失主线因果链，前后矛盾不可自动修复。

**框架预测**：根据原则42（能力边界节点划分）、45-46（上下文审慎注入+分级滚动）、47（外部程序硬约束）、50（写测分离）、55（一步一验），一个按以下方式设计的Agent框架应在长篇叙事连续性上大幅超越通用方案：

1. **流程节点划分**（原则42）：长篇拆为"结构大纲→卷→章→节→场景"五级节点，每级产出经人工确认后再展开下级。单次生成的信息量始终控制在窗口有效容量内。

2. **叙事描写分离+分级上下文管理**（原则45-46）：叙事主线（人物弧光、因果链、伏笔）常驻高优先级上下文，场景描写按当前章节按需滚动注入。叙事和描写在两个通道中分别管理——不共享同一段上下文。

3. **程序硬约束追踪**（原则47）：角色状态、时间线、伏笔库由独立程序逻辑追踪（非模型上下文记忆）。每段生成前程序注入当前状态快照，生成后程序校验一致性。关键分叉点——人物出场、时间跳跃、伏笔回收——全部由程序标记而非模型自判。

4. **多代理写测分离**（原则50 + 推论5）：叙事生成（写）、逻辑一致性审查（测）、文学校对（审）由三个独立Agent实例在独立上下文中执行。写和测不共享上下文——避免自写自验的循环盲区。

5. **一步一验**（原则55）：每个场景完成后即进行一致性校验，不攒到章节末尾。前一场景的未验证输出绝不允许进入下一场景的上下文。

**结果**：实现了通用Agent框架未能达成的长篇无主干叙事逻辑断点。框架的原则42、45-47、50、55在此案例中被联合验证。

**与框架论证的对应**：

| 框架主张 | 案例中的体现 |
|---------|-------------|
| 超边界任务必须节点分解（推导一） | 五级拆分使每级不超有效容量 |
| 上下文不经收敛，容量优先管理（§2.2, §2.4） | 叙事主线常驻+描写按需滚动 |
| 关键分叉须程序硬约束（推导三；结构根因：ℝⁿ 跨层不复合） | 角色/时间线/伏笔由程序追踪，不由模型自判 |
| 写测分离避免自检盲区（原则50） | 写-审-测三Agent独立上下文 |
| 一步一验截断错误传导（原则55） | 场景级校验，非章节级 |

### 6.1 数学基础的未闭合项

#### 6.1.1 不对称闭合——从"弱闭合"到精确结构刻画（全框架最薄弱环节的重新理解）

**v3 的诊断**：
> 理论要求：A_effective = A_L · ... · A_1（矩阵乘法 = Chapman-Kolmogorov 积分）
> 实际计算：x_L = x_0 + Σ(A_l·V_l·W_l^O + MLP_l(...))（加法累积）
> 当前状态：仅可弱闭合——接受功能类比，放弃严格数学对应

**v4 的精炼诊断**（经 §2.1 的 ℝⁿ⊗ℝᵈ 分解和笔记分析）：

这个"不匹配"不是模糊的"加法 vs 乘法"——它是残差流张量积 ℝⁿ ⊗ ℝᵈ 中两个因子的**范畴状态不对称**：

| 因子 | 范畴状态 | 累积方式 | 米田 |
|------|---------|---------|:--:|
| ℝᵈ（语义空间） | 闭合——W 矩阵可复合 | 乘性累积（W_l · ... · W_1） | ✓ |
| ℝⁿ（路由空间） | 不闭合——A 矩阵跨层不可复合 | 无累积（每层重新参数化） | ✗ |
| 整体残差流 | 加法连接（离散 Euler 积分器）——唯一能维持边缘混沌的累积方式 | — | — |

**关键推进**：不对称闭合不是需要修复的缺陷——它是 Transformer 能够产生语义结构的数学前提。
- ℝⁿ 闭合 + ℝᵈ 闭合 → Markov 平稳分布 → 同质化 → 语义死亡
- ℝⁿ 不闭合 + ℝᵈ 不闭合 → 无语义自足结构 → 深度坍缩为单层
- ℝⁿ 不闭合 + ℝᵈ 闭合（实际架构）→ 语义空间有自足结构 + 路由在演化中的语义空间上动态决策 → 语义动力学

**因果耦合的精确陈述**：W 的复合改变语义空间坐标 → A 依赖语义空间 → A 逐层重新参数化 → A 跨层不复合。W 的复合成功 = A 的复合失败的原因。两个因子的闭合状态不对称，且因果互补。

**残差加法的功能角色（重新理解）**：加法不只是"与态射复合不匹配"——它是唯一能长期维持边缘混沌（§2.5）的累积方式。乘法残差将导致轨线指数衰减到 0（深度坍缩）或指数发散（表示爆炸）。加法残差通过 Jacobian = I + ∂F/∂x 使特征值维持在 ≈1 附近 → 边缘混沌可持续 → 深度有持续信息增量。

**当前闭合状态**：
- 结构描述：**严格层**——ℝⁿ 和 ℝᵈ 的范畴状态判定是数学事实
- 功能论证（不对称闭合 = 功能性的）：**形式论证层**——从范畴状态 + 反事实推理推导
- 对"闭合"的放弃：**已大幅收窄但未消除**——从"整个框架最薄弱环节"变为"一个已有精确结构刻画、功能含义已澄清、但强范畴论闭合（如 Effectus/Para 构造）尚未完成的环节"
- 强闭合路径：引入 Effectus 理论或 Para 构造——但复杂度大幅上升，且即使成功也只能描述结构类型，无法给出收敛保证或表达能力的具体预测

**与 v3 的根本区别**：v3 把"弱闭合"当作需要诚实标注的**缺陷**。v4 将其重新理解为 Transformer 架构的**功能性数学结构**——精确刻画了哪些部分闭合、哪些不闭合、为什么不对称、以及为什么这个不对称是语义动力学的前提。论证姿态从"我们这里不够好"变为"我们找到了为什么它必须是这样的"。

**来源**：闭合分析缺口二第二步 + 笔记 01-10 的完整分析链

#### 6.1.2 有限范畴下"截断投影"不构成限制

- 框架自己选择的数学设定（FinStoch、有限 token 集）恰恰是有限维范畴
- 在有限范畴上，米田嵌入本身就是有限维的——"截断投影"作为原则性限制不成立
- 修复方向：将"语义空间"拆为两层——底层 FinStoch（有限维）+ 上层连续语义空间（无限维），截断发生在上层到下层的信息损失中
- 来源：硬伤分析三

#### 6.1.3 交叉熵不蕴含函子性

- 训练目标 L = -Σ log P(next_token|context) 促使匹配条件分布，不促使 F(g∘f) = F(g)∘F(f)
- 修复：不声称梯度逻辑蕴含函子性，改为"收敛给出稳定性，经验上稳定后的解恰好呈现近似函子性质"
- 来源：硬伤分析一

#### 6.1.4 低秩限制与 CoT RL 补全的逻辑互斥

- 第7条：参数空间有效秩只有数百维 → 不足以编码逻辑结构
- 第9条：CoT RL 补全了逻辑能力
- 两命题不能同时为真。二选一：
  - 路径A：保留悲观诊断，CoT RL不是在增加表征能力而是在优化路径选择
  - 路径B：保留乐观推论，是预训练目标没引导参数表征逻辑结构而非容量不够
- 来源：硬伤分析二

#### 6.1.5 Markov 核捕获语义关系的边界（闭合分析缺口三）

- 可捕获：分布性语义关系（共现、联想、上下文替换）——天然是概率的
- 需论证：组合语义（"红苹果"="红"+"苹果"）——通过态射复合编码；结构化关系（部分-整体、因果）——需范畴中的图表结构
- 不可闭合：规范性关系（逻辑必然 vs 经验高概率）——"概率为1"和"逻辑必然"在 Markov 范畴内不可区分；视角依赖关系（指称、意向性）——依赖认知主体视角，无法被主体中性的 Markov 核完全捕获
- 来源：闭合分析缺口三

#### 6.1.6 边缘混沌假说的经验验证缺口（v4 新增）

- λ_max ≈ 0 的预测（T8）尚未在任何已发表工作中得到直接验证
- 混沌三条件（路径依赖 + 非线性折叠 + 局部扩张）的"同时成立"判断来自对 Transformer 计算图的逐项数学分析，不是数值测量
- 风险：实际 λ_max 可能系统性偏离 0，此时需要解释为何 Transformer 仍能工作——可能"边缘"条件比 λ_max ≈ 0 更宽松
- 风险：残差强度调节（T9）在真实模型中可能因 LayerNorm 的补偿效应而无法产生预测的分叉行为
- 当前状态：诚实标注为概念构造层，T7-T10 的检验结果将决定该假说是升级为形式论证层还是被否证

### 6.2 内部逻辑张力

#### 6.2.1 "无自发动机"未给出根因

- 框架观察到模型没有自发求知动机（草稿29），但未从数学结构中推导出此缺失
- 参数冻结解释了"不能持续学习"，但不能解释"没有学习动机"
- 当前状态：诚实标注为"仅观察未给出根因"
- 来源：草稿57条逻辑句校准

#### 6.2.2 多模态判断需限定范围

- "其他模态只是形式对齐或语言模拟"在2025年显得过于绝对
- GPT-4o/Gemini 2.5的原生多模态是联合训练而非简单对齐
- 需区分：多模态是否有效（工程事实）vs 多模态是否触及人类认识本质（哲学主张）
- 来源：对比分析三

### 6.3 发散框架的精度缺口

- 收敛框架有形式化推导（P1-P6, B1-B3 → 四个推导 + 三段论）
- 发散框架的核心洞察（模型自发采样+偏好信号驱动）已达到概念精度，但整体推导密度低于收敛框架
- 开放问题一：偏好信号的信息论量化刻画（类比验证信号覆盖度的 C_V 条件）
- 开放问题二：发散过程中"新语义与关联"如何从条件概率中涌现——当前框架将发散任务刻画为已有语义空间的偏好引导重组，但草稿第25条暗示发散过程具有生成性（产生全新语义）。这一张力未解决——若模型只能自回归采样已有分布，"全新语义"的涌现机制是什么？
  **[留空——待作者补充实证：发散任务中观察到的新语义涌现现象]**
- **v4 补注**：发散任务的动力学可能与 §2.5 的混沌通路分析有深层联系——"新语义"的涌现可能对应于 ℝᵈ 语义空间中混沌轨线进入新的吸引域。但此联系目前仅为方向性提示，尚未展开

### 6.4 混合任务的复杂性

- 当前仅识别了三种混合模式（序列/层级/并行）
- 未系统分析每种模式下的具体Harness设计约束和任务完成率下界
- 混合模式切换的代价量化未完成
- **[留空——待作者补充实证：发散→收敛过渡的工程案例分析（长篇网文创作框架的结构大纲阶段即属此类过渡——从发散创意到可验证章节标准结晶）]**

### 6.5 "不存在中期记忆"论断的操作化

- 当前论证：长期和短期记忆属于不同数学范畴 → 不存在连续过渡
- 关键未解问题："不存在"是在本体论层面还是工程层面？
  - 本体论→哲学主张，无法在Transformer架构内证伪
  - 工程→可操作化为"上下文信息无法获得参数记忆的结构稳定性"，可检验
- 建议操作化：若某种"中期记忆"方案能展示时间性动态（衰减、巩固、自发重组），则论断在工程意义上被否证
- **补注——推论2（Fine-tuning不能修复上下文不可靠）与推论3（RAG不创造新记忆结构）**：两条推论均从记忆二分直接推出——Fine-tuning改变参数（长期记忆），无法触及上下文不经收敛的结构事实；RAG检索到的文档注入上下文后即成为短期记忆，管理和优化的是"哪些信息进入窗口"而非创造新的记忆结构。当前框架未将两条推论作为独立章节展开——建议在 §2.4 或本文中给予显式陈述

### 6.6 Agent Scaling Law 的精确预测形式

- C6 目前是方向性预测（"更早进入平台期"）
- 需要精确化：平台期差距的量级估计、拐点的任务复杂度定位、与模型规模的关系
- 这决定 C6 是可检验的科学预测还是不可证伪的模糊断言

### 6.7 开放方向

1. **将测试收敛+层级错误传播+根需求外在形式化为可操作的Agent可靠性理论**——这是工程相关性最强、与主流话语对接最容易的方向
2. **范畴论/米田内核的形式化**——可作为深层数学基础事后补充，不必作为第一刀。需与 [Categorical Invariants of Learning Dynamics](https://arxiv.org/abs/2510.04376) (arXiv 2025.10) 等最接近的数学工作进行对话——该工作将学习定义为函子 L: Param→Rep，但未涉及米田引理
3. **偏好信号的量化理论**——收敛框架有 C_V 条件，发散框架需要等价的分析工具
4. **架构改进方向**——如果当前架构的数学结构决定了这些边界，那么什么架构改变可以移动边界？（推理时持续学习、非 softmax 注意力、时间性记忆结构、可复合路由机制）
5. **与 Hippocampus Revolution / Memory Taxonomy (2025) 的对话**——草稿的"无中间记忆"论断构成了对这些工作的根本性挑战，需在学术对话中明确"不存在"的操作化含义（§6.5）
6. **与 Stanford MACI / SocraSynth (2020-2025) 等九柱多Agent理论的对接**——这些工作覆盖 Agent 层但缺数学深度，框架可补充其理论基础
7. **v4 新增：边缘混沌假说的经验检验**——T7-T10 的检验将决定该假说的命运。其中 T10（强制复合→同质化）是最强的否证实验
8. **v4 新增：ℝⁿ 端路由的量化理论**——当前分析确立了 ℝⁿ 不复合是数学事实，但尚未给出路由偏离复合程度的量化度量。笔记提出的"A^(l+1) 偏离 CK 条件的程度"是可能的方向
