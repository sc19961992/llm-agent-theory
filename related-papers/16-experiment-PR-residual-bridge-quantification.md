# 数值实验PR：残差加性累积→乘性复合的逼近精度量化

## 摘要

**待检验假说：** Transformer残差流中的纯加性累积在功能上逼近层间乘性复合。机制证明（toy case）已确立桥接的数学存在性；本实验量化该桥接在真实训练模型中的逼近精度及其随层间距、模型规模和功能模块的分布。

**实验类型：** 计算实验（已训练模型的前向传播分析+消融），无需训练。

---

## 1. 背景与动机

### 1.1 理论背景

Transformer残差流以纯加法累积信息：$x_{\ell+1} = x_\ell + F_\ell(x_\ell)$。在直观上，加法不能"等于"乘法——两个逐层累加的向量不会自动相乘。然而，线性代数揭示了一个机制（toy case已证）：任意两个非相邻层之间存在隐式的"虚拟权重"$W_I^{\ell_r} \cdot W_O^{\ell_w}$——它不在架构中作为参数存在，而是从"前一层的写入矩阵$W_O$"和"后一层的读取矩阵$W_I$"的矩阵乘积中涌现。分配律+结合律保证：

$$W_I^{\ell_r} \cdot x_{\ell_r} = W_I^{\ell_r} \cdot x_0 + \sum_{\ell=0}^{\ell_r-1} (W_I^{\ell_r} \cdot W_O^\ell) \cdot h_\ell$$

即残差流中的加性累积在数学上等价于各层通过虚拟权重的乘性直接连接之和。

### 1.2 机制证明回答不了的问题

上述证明是**对架构的定理**——它证明桥接存在，不依赖任何特定训练参数。但它不回答：

- **Q1（利用度）：** 真实训练后的模型在多大程度上实际利用了这种虚拟权重进行跨层通信？
- **Q2（衰减律）：** 逼近精度如何随层间距$k$衰减？中间层的写入噪声是否使理论预测值与实测因果贡献的偏差随$k$单调增加？
- **Q3（涌现律）：** 较大模型和较深层是否表现出更精确的乘性桥接——即模型在训练中"学会"了利用虚拟权重？

### 1.3 对论文框架的意义

本文§2.7接受"加性累积在功能上逼近乘性复合"的弱闭合立场。本实验为该立场提供经验量化支撑——将弱闭合从诚实标注升级为有经验量化支撑的诚实标注。实验产出的一条$k \to \bar{\varepsilon}(k)$曲线直接作为§2.7的证据锚点。

## 2. 实验设计

### 2.1 模型与数据

| 项目 | 配置 |
|------|------|
| 主模型 | GPT-2 Small（12层，d=768，124M参数） |
| 对照模型 | GPT-2 Medium（24层，d=1024，355M参数）——可选，做参数规模对照 |
| 数据 | The Pile验证集，随机抽样5000条，每条截取128 token |
| 工具 | TransformerLens（hook残差流、缓存激活值）、PyTorch |
| 硬件 | 单GPU（A100或等效），无需多卡 |

### 2.2 层对采样

GPT-2 Small有12层，全部 $\binom{12}{2} = 66$ 个有序层对 $(\ell_w, \ell_r)$，$\ell_w < \ell_r$。全量计算，不采样。

### 2.3 核心指标定义

对每个层对 $(\ell_w, \ell_r)$，定义两个量：

**2.3.1 虚拟权重的理论预测贡献**

$$C_{\text{pred}}(\ell_w, \ell_r) = \frac{\|W_I^{\ell_r} \cdot W_O^{\ell_w}\|_F}{\|W_I^{\ell_r}\|_F \cdot \|W_O^{\ell_w}\|_F}$$

采用归一化Frobenius范数（即两个矩阵乘积的范数除以各自范数的乘积），以消除不同层矩阵尺度差异的影响。该值的范围在$[0,1]$之间：0表示$W_I^{\ell_r}$和$W_O^{\ell_w}$的行/列空间完全正交（写入对读取无理论贡献），1表示完全对齐。

**2.3.2 消融实测的因果贡献**

对每个输入序列，执行消融协议：

```
1. 正常前向传播至层 ℓ_w 结束，记录残差流写入：
   Δ_w = x_{ℓ_w+1} - x_{ℓ_w}   （即该层MLP+Attention写入的总和）

2. 构造消融残差流：
   x'_{ℓ_w+1} = x_{ℓ_w} + Δ_w^⊥
   其中 Δ_w^⊥ 是 Δ_w 在 W_I^{ℓ_r} 行空间上的正交投影的补
   （即：保留不会通过W_I^{ℓ_r}影响层ℓ_r读出的分量，置零会通过W_I^{ℓ_r}影响读出的分量）

3. 继续前向传播至层 ℓ_r，记录：
   y_normal = 层ℓ_r的残差流写入（正常计算）
   y_ablated = 层ℓ_r的残差流写入（消融计算）

4. 对单个序列，实测贡献：
   c_obs(ℓ_w, ℓ_r; seq) = ‖y_normal − y_ablated‖₂
```

对全部5000条序列取平均：
$$C_{\text{obs}}(\ell_w, \ell_r) = \mathbb{E}_{\text{seq}}[c_{\text{obs}}(\ell_w, \ell_r; \text{seq})]$$

**2.3.3 逼近误差**

$$\varepsilon(\ell_w, \ell_r) = \frac{|C_{\text{obs}}(\ell_w, \ell_r) - C_{\text{pred}}(\ell_w, \ell_r)|}{C_{\text{pred}}(\ell_w, \ell_r) + \eta}$$

其中 $\eta = 10^{-6}$ 防止分母为零（当$C_{\text{pred}} \approx 0$时该层对的理论贡献可忽略，逼近误差无意义）。

**2.3.4 $k$-聚合误差**

将层对按间距 $k = \ell_r - \ell_w$ 分组，计算每组均值：

$$\bar{\varepsilon}(k) = \frac{1}{N_k} \sum_{\ell_r - \ell_w = k} \varepsilon(\ell_w, \ell_r)$$

其中 $N_k = L - k$（对于GPT-2 Small，$L=12$）。$\bar{\varepsilon}(k)$ 是本实验的核心产出曲线。

### 2.4 消融协议的技术细节

**正交投影分解：** 将 $\Delta_w \in \mathbb{R}^d$ 投影到 $W_I^{\ell_r}$ 的行空间上。对 $W_I^{\ell_r} \in \mathbb{R}^{d_h \times d}$ 做SVD分解：$W_I^{\ell_r} = U \Sigma V^T$，取其右奇异向量中前$r$个显著分量（$r$选择使得累计方差贡献≥95%）构成投影子空间的基。则：

$$\Delta_w^\parallel = \sum_{i=1}^r (v_i^T \Delta_w) \cdot v_i$$
$$\Delta_w^\perp = \Delta_w - \Delta_w^\parallel$$

消融时将 $\Delta_w^\parallel$ 置零（仅移除$\Delta_w$中会通过$W_I^{\ell_r}$影响层$\ell_r$读出的分量），保留$\Delta_w^\perp$（$\Delta_w$中与层$\ell_r$读出无关的分量——对其他下游层可能有用，不进行不必要的破坏）。

**对注意力头粒度的细化（可选扩展）：** 上述协议将整个层的写入作为一个整体消融。更精细的方案是将$\Delta_w$按注意力头和MLP分解，对每个写入源独立计算$\Delta_w^\parallel$。当需要区分"不同功能模块的写入是否在正交子空间中"（检验Q3）时，可采用头粒度分解。

### 2.5 基线：随机配对对照

将$W_O^{\ell_w}$和$W_I^{\ell_r}$的层索引随机打乱，重新计算$C_{\text{pred}}^{\text{shuffled}}$和对应的消融。若真实层对的$C_{\text{obs}} / C_{\text{pred}}$比值系统性高于打乱基线，则表明模型在训练中"有意"对齐了特定层对的读写子空间——虚拟权重的乘性桥接是学习的产物，不仅是架构的线性代数副产物。

## 3. 可检验预测

### 预测1：层间距衰减律

$$\bar{\varepsilon}(k) \text{ 随 } k \text{ 单调递增}$$

**机制：** 中间$k-1$层各自独立写入残差流。在$\ell_w$写入后、$\ell_r$读取前，残差流中的信息被$k-1$次中间写入污染。$C_{\text{pred}}$假设各层写入是独立可分解的（虚拟权重是将$W_I^{\ell_r} \cdot x_{\ell_r}$展开为各层贡献的**和**——求和项之间的交互在代数上仅当残差流保持严格线性时被保留；MLP的非线性和LayerNorm的归一化破坏了这一假设）。$k$越大，中间非线性操作累积越多，$C_{\text{obs}}$偏离$C_{\text{pred}}$的幅度越大。

**操作化判据：** $\bar{\varepsilon}(k)$ 的Spearman秩相关系数 $\rho(k, \bar{\varepsilon}) > 0$ 且统计显著（$p < 0.05$）。

**若预测被否证（$\bar{\varepsilon}(k)$ 大致平坦）：** 说明中间层的非线性写入并未系统性累积为逼近误差——可能原因：(a) 层间写入子空间高度重叠（写入在同于空间中叠加而非分散），(b) MLP/LayerNorm的击穿效应被注意力在读取侧进行了自适应补偿，(c) 模型在训练中学会了一种"隐式误差校正"机制（类似于各层自适应地补偿前序层的非线性偏差）。这些可能性本身构成新的研究问题。

### 预测2：深层涌现律

对GPT-2 Small：

$$\bar{\varepsilon}_{\ell_w > 6}(k=1) < \bar{\varepsilon}_{\ell_w \le 6}(k=1)$$

$$|\bar{\varepsilon}_{\text{GPT-2 Medium}}(k=1,2) - \bar{\varepsilon}_{\text{GPT-2 Small}}(k=1,2)| > 0 \text{（前者更小）}$$

**机制：** GPT-2 Small在第6层以上出现induction head——一种已知的跨层电路。induction head的运作依赖K-composition（前一层的键矩阵$W_K$与后一层的查询矩阵$W_Q$的虚拟权重乘积）。如果浅层（≤6层）的特征尚未被训练充分分化，$W_O$和$W_I$的读写子空间重叠度高，$C_{\text{pred}}$和$C_{\text{obs}}$均应较高（但精度差异来自重叠而非有意对齐）。深层（>6层）应呈现相反的模式：$C_{\text{pred}}$高（有意对齐）+ $C_{\text{obs}}$高（实际贡献大）+ $\varepsilon$低（逼近精度高）——这表明模型**在特定功能模块中利用而非偶然触发**虚拟权重的乘性桥接。

**操作化判据：** 对$\ell_w > 6$的层对，$\varepsilon$在$k=1,2$处显著低于$\ell_w \le 6$层对的对应值（Welch's t-test, $p < 0.05$）。对GPT-2 Medium验证时，$\bar{\varepsilon}(k=1,2)$系统低于GPT-2 Small。

### 预测3：功能模块特异化

**方法：** 复制Anthropic的induction head检测协议（在GPT-2 Small中已知induction head位于第5,6,7层的部分注意力头）。标记出这些head所在的层索引。构造两组层对：

- **功能组：** $(\ell_w, \ell_r)$ 中至少一层包含已知induction head
- **对照组：** 全部层对中排除功能组后的剩余

**预测：** 功能组的 $\bar{C}_{\text{obs}}$ 高于对照组，且功能组的 $\bar{\varepsilon}$ 低于对照组。

**操作化判据：** Mann-Whitney U检验，$p < 0.05$。

## 6. 诚实标注

1. **消融的近似性：** 将$\Delta_w$精确分解为"会通过$W_I^{\ell_r}$影响读出"和"不会影响"两个分量时，使用了$W_I^{\ell_r}$的SVD截断（95%方差）。截断丢弃的5%方差在高维空间中可能仍携带可检测的信号——$C_{\text{obs}}$稍有低估。实验报告需注明显著奇异值的数量$r$及其方差覆盖比例。

2. **单层写入 vs 头粒度写入的取舍：** 按整个层（而非按注意力头+MLP）分解写入，忽略了同一层内不同头可能写入正交子空间的情况。若某层的head A写入子空间S_A，head B写入子空间S_B，且S_A ⊥ S_B，但两者均与$W_I^{\ell_r}$的行空间有非零交集——则以层为单位的消融会将两者同时移除，夸大$C_{\text{obs}}$（高估了"该层写入"对读出的总因果贡献）。头粒度消融（见2.4节可选扩展）可以分离此效应，但执行成本×H倍（H为头数）。本实验默认采用层粒度方案；若预测2/3的检验结果接近显著阈值，应补充头粒度消融以排除假阳性。

3. **归一化方案对结果的敏感性：** $C_{\text{pred}}$的归一化公式（除以$\|W_I\|_F \cdot \|W_O\|_F$）消除了不同层矩阵范数尺度差异的混淆——但它也消除了"范数本身所携带的信息量"。例如，若GPT-2 Medium的$W_O$和$W_I$矩阵的Frobenius范数系统性大于GPT-2 Small，则同等对齐度下（$C_{\text{pred}}$归一化值相同），GPT-2 Medium的原始因果贡献$C_{\text{obs}}$应更大——因为写入和读出矩阵的能量更大。实验报告应同时给出归一化和未归一化的指标作为对照。

4. **因果归因的局限：** $C_{\text{obs}}$度量的是"移除层$\ell_w$写入后，层$\ell_r$输出的变化量"——这是因果效应（causal effect），但不等价于"层$\ell_w$向层$\ell_r$传递的信息量"。因果效应包含直接效应（写入→读出）和间接效应（写入→改变中间层行为→间接影响读出）。在Transformer的残差结构中，这两种效应在消融协议中被同时移除——$C_{\text{obs}}$可能高估直接通信量。通过分析$k=1$时的$C_{\text{obs}}$（此时无中间层，间接效应为零）作为基线，可以近似分离直接效应。
