# 加性累积何以逼近乘性复合：一个数值toy case

## 问题

Transformer的残差流在每一层只做纯加法：x ← x + F(x)。直觉上这似乎排除了层间的乘性交互——两个加法不会自动"相乘"。但Anthropic的Transformer Circuits框架指出，残差流的线性性质使得**任意两个非相邻层之间的读写交互等价于一个隐式矩阵乘积**（"虚拟权重"）。这个toy case用2×2矩阵演示该机制。

## 设定

2维残差流。层1先写入信息，层2后读取。两者之间可以有任意多层——为简洁取两层。

- 残差流初始值：x₀ = [0, 0]^T
- 层1的注意力头输出向量：h₁ = [3, 1]^T（这是注意力计算的结果，具体值来自上游）
- 层1的输出投影矩阵：W_O¹ = [[1, 0], [0, 2]]
- 层1写入残差流：x₁ = x₀ + W_O¹·h₁ = [0,0]^T + [3, 2]^T = [3, 2]^T

现在残差流中存储了 [3, 2]^T。中间可以插入任意多层（它们各自读写但不覆盖层1的信息——残差流的加性确保各层写入在流中共存）。

- 层2的输入投影矩阵：W_I² = [[0.5, 0], [0, 1.0]]
- 层2从残差流读取：W_I²·x₁ = [[0.5,0],[0,1]]·[3,2]^T = [1.5, 2.0]^T

## 关键展开

把层2的读操作展开到层1的写操作：

```
W_I² · x₁
= W_I² · (x₀ + W_O¹ · h₁)          [残差流的定义]
= W_I² · x₀ + W_I² · W_O¹ · h₁    [矩阵乘法的分配律]
= W_I² · x₀ + (W_I² · W_O¹) · h₁
```

第二项中的 **W_I² · W_O¹** 就是"虚拟权重"——一个2×2矩阵，它直接连接了层1的注意力输出h₁和层2的输入。计算出来：

```
W_I² · W_O¹ = [[0.5, 0], [0, 1.0]] · [[1, 0], [0, 2]]
            = [[0.5×1 + 0×0, 0.5×0 + 0×2],
               [0×1 + 1.0×0, 0×0 + 1.0×2]]
            = [[0.5, 0], [0, 2.0]]
```

这个矩阵在架构中不存在——没有任何一层的参数是这个矩阵。但它从层1的写入和层2的读取的**组合**中涌现出来。

## 数值验证

架构实际计算（两层各自做加法）：
```
x₁ = [0,0] + [1,0; 0,2] · [3,1] = [0,0] + [3,2] = [3,2]
层2读：W_I² · x₁ = [0.5,0; 0,1] · [3,2] = [1.5, 2.0]
```

等价于存在一条直接连接（乘性复合）：
```
h₁ 通过虚拟权重 [[0.5, 0], [0, 2.0]] 直接贡献到层2的输入
贡献量 = [[0.5, 0], [0, 2.0]] · [3, 1]^T = [1.5, 2.0]^T
```

两条路径给出相同的数值结果。**这就是加性累积逼近乘性复合的机制：加性发生在残差流中，乘性发生在不同层的读写投影矩阵之间。** 层间交互在数学上是乘性的，但架构上不需要任何显式的乘法门控——它从"先写后读"的线性组合中自然产生。

## 扩展到多层

在第ℓ层和第ℓ+k层之间，如果中间没有任何层修改层ℓ写入的信息（这在残差流中通常不成立——中间层也会写入——但当不同层写入不同的低维子空间时近似成立），虚拟权重推广为：

W_I^{ℓ+k} · W_O^ℓ

如果中间层也有写入，则路径展开定理（Anthropic, 2023）给出：整个模型的前向计算可以写成所有可能路径上的矩阵乘积之和。每条路径经过若干层的W_O，在最后一层通过W_I读出。各路径内部是乘性复合（矩阵连乘积），路径之间是加性累积（残差流将它们加在一起）。

## 局限

这个机制解释了为什么**加性能够工作**——因为层间的读取天然引入了乘性。但它没有解释为什么加性**不能严格等价于乘性**——因为：

1. 不同层在残差流中写入的子空间可能重叠，导致信息交叉污染而非纯乘性复合
2. 中间层的非线性（MLP、LayerNorm）打断了严格的线性分配律
3. 残差流只有加法，没有任何机制（除了权重学习本身）来保证旧信息在应该被替换时真正被"覆盖"而非"叠加"

第三个局限正是Deep Delta Learning试图通过显式门控来解决的问题：当需要精确覆写而非叠加时，纯加性残差无法在架构层面保证操作的精确性。

## 与本文框架的衔接

该toy case演示了§2.7中"加性累积在功能上逼近乘性复合"这一弱闭合主张的具体机制。加性⇔乘性的桥接不是神秘的涌现——它是残差流线性性质与矩阵乘法分配律的直接后承。桥接存在，但严格等价不成立（DDL的β=1是架构层面的保证，纯加性残差中没有对应物）。两项独立研究（Anthropic和DDL）从"为什么能工作"和"为什么有限"两个方向为本文的弱闭合诚实标注提供了数值层面的直观支撑。
