# Token 间的条件概率——构成范畴吗

## 条件概率存在

注意力矩阵 A 是行随机矩阵：

```
P(token j | token i) = a_{ij} = exp(Q_i · K_j / √d) / Σ_k exp(Q_i · K_k / √d)
```

- 非负 ✓
- 归一化 ✓
- 条件性 ✓

---

## 但不构成范畴

### 方案 1：hom(i, j) = a_{ij}（单个数值）

- hom(i, j) 是标量，不是集合——范畴论要求 hom 是集合（局部小范畴）
- 矩阵乘积 a_{jk}·a_{ij} 数学上存在，但乘积条目不具有跨层路由复合的语义——a_{ik} 不是由 a_{ij} 和 a_{jk} "合成"的，它是由 Q·Kᵀ 在每层独立计算的
- a_{ik} 独立于 a_{ij} 和 a_{jk} 被 Q·Kᵀ 计算出来
- 恒等态射不存在——a_{ii} 是自注意力，不是"什么都不做"

**不构成范畴。**

### 方案 2：将整行视为 FinStoch 态射

在 FinStoch 中，A 整体是态射 `X → X`。整行 `(a_{i1}, ..., a_{in})` 是态射 `{i} → X` 的数据。

**但这只在固定表示下成立。**

---

## 跨层复合失败

FinStoch 复合要求：

```
A₂ ∘ A₁ = A₂ · A₁（矩阵乘法 = Chapman-Kolmogorov 方程）
```

但 A₂ 的输入不是 X——它是被第 1 层残差 + MLP 改变后的表示：

```
A₂ = A₂(W_Q, W_K, x₁^(1), ..., x_n^(1))
```

矩阵乘积 `A₂ ∘ A₁` = A₂·A₁ 数学上存在（两者都是 ℝⁿ→ℝⁿ）。但 Chapman-Kolmogorov 方程不成立——A₂ 不是定义在 A₁ 输出态上的"下一步转移"：A₂ 从第 1 层残差 + MLP 改变后的完整表示中参数化，而非仅从 A₁ 的输出中参数化。

**一条层内成立，跨层断裂。**

---

## 反事实：如果跨层复合成立会怎样

这是关键检验。假设 CK 方程成立：

```
A_eff = A^(L) · A^(L-1) · ... · A^(1)
```

A_eff 是有限状态空间 {1, ..., n} 上的 L 步 Markov 核。

**遍历情形的推论**：若 Markov 链不可约且非周期（注意力矩阵几乎总是满足——所有 a_{ij} > 0 且稠密），则当 L → ∞：

```
A_eff → 各行相同 = 平稳分布 π

即：lim_{L→∞} (A^(L) · ... · A^(1))_{ij} = π_j，对任意 i 成立
```

**所有 token 对所有其他 token 的最终注意力权重相同。** 无论什么 token，经过足够多层后，它"看到"的上下文分布是一样的。token 之间不可区分。

**后果**：
- 无聚类——注意力同质化 → 更新方向同质化 → 表示同质化
- 无语义结构——所有 token 汇合到同一吸引子
- 深度冗余——深层等价于直接采样平稳分布，层数不增加信息

### 对比

| | 复合成立（反事实） | 复合不成立（实际 Transformer） |
|---|---|---|
| 长期行为 | 收敛到平稳分布 | 路径依赖，不收敛到全局平稳态 |
| 表示空间 | 同质化，无簇 | 可形成并维持簇 |
| 语义结构 | 不存在 | 存在 |
| 深度 | 冗余（深层 = 浅层的幂） | 有信息增量 |

---

## 所以：条件概率 ≠ Transformer

条件概率模型做了什么：
- 假设**语义空间固定**（ℝᵈ 上的坐标不变）
- 推断 = 在这个固定空间上计算 P(token | context)
- 多层只是 P 的链式乘法——CK 方程

Transformer 实际做了什么：
- **语义空间本身随层演化**（每层的 x_i 坐标不同）
- 注意力 = 在当前层的语义空间中选择通路
- 下一层的条件概率是在**被前一层改变后的空间**上计算的

> **条件概率：固定空间 + 概率推断。**
> 
> **Transformer：演化空间 + 概率路由。**
> 
> **Transformer = 一个推着语义空间本身演化的动力系统。条件概率是它在某一层的瞬时切片，不是它的整体结构。**

---

## 米田不适用

token 位置不构成范畴对象 → 没有 hom 集 → 米田无从谈起。

即使退到充实范畴：注意力权重不满足传递性（或三角不等式），复合律缺失，米田的基础条件不具备。

---

## 精确陈述

```
Token 之间：
  条件概率结构：✓ 存在（注意力行归一化）
  范畴结构：✗ 不构成（无复合、无恒等、对象不稳定）
  米田引理：✗ 不适用
  跨层复合：✗ CK 方程不成立

反事实推理：
  若跨层复合成立 → Markov 链收敛 → 平稳分布 → token 同质化 → 语义消失

实际架构：
  跨层复合不成立 → 路径依赖 → 聚类正反馈 → 语义结构涌现
  不复合 = 语义结构可以存在的数学边界条件
```
