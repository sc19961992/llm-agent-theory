# 命名备选

## 核心发现

> Transformer = L × H 代嵌套循环：每代 = 米田骨架 → 非米田函子（降维·重路由·阻断）→ 外米田骨架（= 下一代的起点）。纯米田 → 坍缩，纯非米田 → 退化。生效域在两根边界之间。

## 命名方向

### 中文

| 名称 | 优势 | 劣势 |
|---|---|---|
| **嵌套米田机制** | 直截，逻辑链完整 | "米田"需领域知识 |
| **层间遗传机制** | 强调残差流累积，易理解 | 丢掉了范畴论来源 |
| **米田循环假说** | 假说程度恰当 | 可能被指不够硬 |
| **内外米田生成** | 精确描述结构 | 过长 |
| **骨架-变异循环** | 直觉形象，好翻译 | 少了范畴论标记 |
| **逐层米田复归** | 更接近数学描述 | 陌生词多 |
| **米田-反米田交替机制** | 强调对立统一 | 过长 |

### 英文

| Name | Note |
|---|---|
| **Nested Yoneda Cycle** | Direct translation, clear structure |

## 论文中的建议用法

- **主要命名**：嵌套米田机制（Nested Yoneda Mechanism）
- **简称**：层间遗传（Layered Inheritance）
- **首次出现时**：完整写出"嵌套米田循环——每层完成一次内米田骨架到外米田骨架的非米田函子映射"
- **范畴论精确定位**：非自治 pointed endofunctor 族的层间动力系统，带终端 exact 函子投影
- **通俗版**：Transformer 的每一层都在继承上层的全部信息（骨架），在上面做一次新的推理路由（变异），然后把增强后的信息传给下一层。L 次循环后，残差流上累积了 L×H 条推理通路。
