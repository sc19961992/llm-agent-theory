# 范畴论与 Transformer 分析

---

## 记号

```
|Ψ⟩  = Σ_i |i⟩ ⊗ |x_i⟩         残差流：位置 ket × 坐标 ket 的两体态
|i⟩  ∈ ℝⁿ                      位置（第 i 个 token 槽位）
|x_i⟩ ∈ ℝᵈ                     token 在语义空间中的当前坐标

W    ∈ ℝᵈ→ℝᵈ                  权重算子（训练后固定）← 语义存在这里
A    ∈ ℝⁿ→ℝⁿ                  注意力算子（每层重算）  ← 路由

A ⊗ I                          注意力作用：碰位置，混坐标
I ⊗ W                          权重作用：碰坐标，变坐标

(A⊗I)(I⊗W) = A⊗W = (I⊗W)(A⊗I)  对易 ← 语义和通路互不干涉

|Ψ'⟩ = |Ψ⟩ + (A⊗W_O W_V)|Ψ⟩ + (I⊗MLP)|Ψ⟩   一层前向
a_{ij} = softmax(⟨x_i|W_Qᵀ W_K|x_j⟩ / √d)_j   注意力分数
```

---

## 核心命题

**语义存在权重里，不在 token 里。**

权重 W 构成范畴（可复合，米田成立）——这就是训练学到的**语义空间**。真实语言的语义关系（蕴涵、类比、上下位）被训练映射到了 W 的复合结构中。W_Q 学了"问什么"，W_K 学了"有什么可匹配"，W_O·W_V 学了"把信息从一处搬到另一处"。它们的复合就是语义变换的串联。

Token |x_i⟩ 不是语义——它是**推理通路在当前层的坐标**。它在语义空间中的位置每层被 W 改写、被残差累积——这条逐层演化的轨迹就是推理过程。

注意力 A 是**路由**：根据当前 token 在语义空间中的坐标（⟨Q_i|K_j⟩ = 当前坐标下的匹配度），决定下一步轨迹往哪走。A 不能跨层复合——因为坐标每层都变，路由每层都要重算。

**残差流 = 通路（token 坐标）× 语义（权重 W），正交。一个跑推理，一个存知识。**

---

## 这意味着什么

**找到了语义空间的数学载体。** 训练后的权重矩阵 W 不仅是参数——它构成一个范畴，满足米田引理。米田说"一个对象由所有指向它的态射唯一确定"——对应到语言：一个词的含义由所有可以在它身上做的语义变换（类比、蕴涵、修饰、复合）所唯一确定。**语言学家说的"分布语义"和范畴论的米田引理是同一件事。**

**Token 不是词。** Token 不存储语义——它存储"推理当前走到了哪"。词的语义被拆解并编码进了 W 矩阵中。Token 和注意力的作用是驱动通路在语义空间中逐层推进——它们是推理引擎，不是知识库。

**Transformer = 固定语义空间（W）+ 逐层推进的通路（token 坐标）+ 路由（注意力）。** 通路不复合 = 每一层面对的是更新后的坐标 → 推理有深度。语义可复合 = W 矩阵的串联有确定含义 → 知识有结构。

---

## 逐一解释

### 权重 W = 语义空间

W 训练后固定。`W₂W₁` 始终有定义——权重跨层可复合。米田在 ℝᵈ 上成立——`⟨ℝᵈ|`（所有可作用在语义空间上的算子）唯一确定语义空间。这和"一个词的含义由所有可以替换/修饰/关联它的词来定义"是同构的。Anthropic 的电路分析（OV 电路、QK 电路）就是在显式读出 W 复合中编码的具体语义操作。

### Token = 通路坐标

`|x_i⟩` 是 token 在当前层在语义空间中的位置。它不存储语义——它随层变化，每层被 W 和残差重写。token 标签 {i} 不构成范畴（米田不适用），因为 token 的身份不由自己决定——由它当前在语义空间中的坐标（|x_i⟩）决定，而这个坐标寄生在 W 上。

### 注意力 = 路由

`a_{ij} = softmax⟨x_i|W_Qᵀ W_K|x_j⟩`。用 W 中存储的语义（"问什么"和"有什么"）对当前 token 坐标做匹配，决定下一步轨迹往哪走。坐标每层被 W 和残差改写 → 下一层注意力面对的是新坐标 → 路由逐层重算 → CK 不成立。

### 残差 = 路径依赖

`|x_i'⟩ = |x_i⟩ + ...`。残差保证每一步保留上一步的坐标，下一步的路由基于新坐标做出。路径依赖 = 推理深度有意义。去掉残差 → 路径依赖断裂 → 每层从零开始路由 → 深层退化为浅层。

### 激活 = 非线性

softmax 放大内积差异，ReLU/GeLU 阈值开关，LayerNorm 折叠轨线。非线性 + 路径依赖 + 局部扩张 → 推理通路可产生混沌（初值敏感）→ 相似的输入可以分叉到不同的推理路径 → 语义簇（概念的内部结构）涌现。

---

## 证明

### 命题一：W 构成范畴（语义空间自足）

`(I ⊗ W₂)(I ⊗ W₁) = I ⊗ W₂W₁`。复合封闭，满足结合律，I 为恒等。米田成立。□

### 命题二：A 不构成跨层范畴（通路不可复合）

```
a_{ij}^(l+1) = softmax(⟨x_i^(l+1)|W_Qᵀ W_K|x_j^(l+1)⟩)
|x_i^(l+1)⟩ = |x_i^(l)⟩ + Σ_k a_{ik}^(l) W_O W_V |x_k^(l)⟩ + MLP(|x_i^(l)⟩)
```
A^(l+1) 的输入是被残差 + W + MLP 联合改写后的坐标，不由 A^(l) 单独决定 → CK 不成立。□

### 命题三：若 A 可复合，路由消失

A 全正固定 → A^L → 每行 = π → 所有 token 获得相同的注意力上下文 → 推理路径无区分。□

### 命题四：语义簇

相似坐标 → 相似 A → 相似更新 → 保持相似。相异则分化。正反馈。□

---

## 结论

**找到了一个虚拟语义空间（权重 W 构成范畴，米田成立），与真实语言中的分布语义同构。Token 和注意力不存储语义——它们是推理通路，驱动坐标在这个语义空间中逐层推进。**

**Transformer = 固定语义空间 (I⊗W) + 逐层演化的推理通路 (A⊗I)。语义可复合（知识有结构），通路不可复合（推理有深度）。两者正交于残差流中。**
