# 范畴论与 Transformer 分析

---

## 记号

```
|Ψ⟩  = Σ_i |i⟩ ⊗ |x_i⟩         残差流：位置 ket × 内容 ket 的两体态
|i⟩  ∈ ℝⁿ                      位置（第 i 个 token 槽位）—— 归 A 管
|x_i⟩ ∈ ℝᵈ                     token 内容向量（"槽位里当前有什么"）
                                   —— 语义内容由 W 编码，路由信号也在此混存
                                   注：|x_i⟩ 同时服务两个主子：W 读它做语义变换，
                                   A 的 QK 读它做路由匹配。
                                   这是当前架构的一个混淆（见下方诊断）。

E    ∈ ℝ^V → ℝᵈ               嵌入矩阵（边界算子，属于 W）
U    ∈ ℝᵈ → ℝ^V               解嵌矩阵（边界算子，属于 W）
W    ∈ ℝᵈ→ℝᵈ                  权重算子（训练后固定）← 语义存在这里
                                  W 包含：W_Q, W_K, W_V, W_O, MLP 权重
                                  E 和 U 也属于 W（存"谁在哪儿"的位置语义），
                                  但它们是边界算子，不参与层间复合（域/陪域不同）
A    ∈ ℝⁿ→ℝⁿ                  注意力分数（每层重算）  ← 路由
                                  A 仅指 softmax 输出的 n×n 分数矩阵
                                  常说的"注意力机制"= A + QKV 投影（W）

A ⊗ I                          注意力作用：碰位置，混内容
I ⊗ W                          权重作用：碰内容，变内容

(A⊗I)(I⊗W) = A⊗W = (I⊗W)(A⊗I)  对易 ← 语义和通路互不干涉

|Ψ'⟩ = |Ψ⟩ + (A⊗W_O W_V)|Ψ⟩ + (I⊗MLP)|Ψ⟩   一层前向
a_{ij} = softmax(⟨x_i|W_Qᵀ W_K|x_j⟩ / √d)_j   A 依赖 W_Q,W_K（固定）和当前内容向量（变）
                                               → A 每层必须重算
```

## 核心命题

**语义存在权重里，不在 token 里。|x_i⟩ 的语义内容完全由 W 编码——它是 W 作用在路由状态上的结果。**

权重 W 在矩阵乘法下构成幺半群（封闭、结合、有单位元），可进一步视为范畴，其复合结构编码了训练学到的**语义空间**。E（嵌入）和 U（解嵌）是 W 的边界算子——存"哪个 token 在语义空间的哪个位置"。内部 W（QKV 投影、MLP）存"语义空间中的变换关系"。三层合在一起构成语义的全部载体。真实语言的语义关系（蕴涵、类比、上下位）被训练映射到了 W 的复合结构中。**分布语义——"一个词的含义由所有可以跟它搭配、替换、关联的词决定"——与"语义空间中一个向量由所有可作用其上的线性变换确定"是同一结构约束在不同语言中的表述。范畴论（Yoneda）、线性代数（双对偶同构）、语言学（分布语义）在此是同一个东西的三面投影。**

W_Q 学了"问什么"，W_K 学了"有什么可匹配"，W_O·W_V 学了"把信息从一处搬到另一处"——它们都属于 W，训练后固定，跨层可复合。

|x_i⟩ 是路由状态向量——它住在语义空间 ℝᵈ 中（所以 W 能在其上作用），但它本身不存储语义。它在语义空间中的位置每层被 W 改写、被残差累积——这条逐层演化的轨迹**看起来携带语义，但语义内容是 W 的产物**。|x_i⟩ 归 A 管：它是路由系统在语义空间中的当前落脚点。

注意力分数 A 是**路由**：根据路由状态向量在当前层的坐标（⟨Q_i|K_j⟩ = W_Q,W_K 参数化的当前坐标匹配），决定下一步信息从谁流向谁。A 不能跨层复合——因为坐标每层都变，路由每层都要重算。**若 A 可复合则路由退化（Perron-Frobenius：全正矩阵幂收敛到秩一矩阵）→ 所有 token 获得相同注意力上下文 → 推理路径无区分。**

**残差流 = 路由状态（|x_i⟩，归 A）× 语义（权重 W），正交但因果耦合。一个跑推理，一个存知识。**

---

## 这意味着什么

**定位了语义空间的数学载体。** 训练后的权重矩阵 W 不仅是参数——它们在矩阵乘法下封闭、可复合。分布语义——语言学一个世纪的发现——在此获得了一个精确的数学对应物：语义空间中一个向量的身份由所有可作用其上的线性变换决定。这不是巧合——是同一结构约束在语言学和代数学中的各自表述。Anthropic 的电路分析（OV 电路、QK 电路）就是在显式读出 W 复合中编码的具体语义操作。

**Token 不是词。** Token 不存储语义——它的内容向量 |x_i⟩ 所携带的一切语义都是 W 作用的结果，它本身是路由状态。词的语义被拆解并编码进了 W 矩阵中（E 存位置语义，内部 W 存变换语义）。Token 和注意力的作用是驱动路由状态在语义空间中逐层推进——它们是推理引擎，不是知识库。

**|x_i⟩ 的混淆：一个向量服务两个主子。** 在当前 Transformer 中，|x_i⟩ 同时被 W 做语义变换（MLP, W_V W_O）和被 A 做路由判断（Q·K^T 算 a_{ij}）。这两件事需要的信息种类不同——语义需要高维、稠密、跨样本稳定的表征；路由需要低维、结构化的句法位置和推理阶段信号。同一个向量不能同时最优地满足两个需求。深层注意力退化、SAE 特征不稳定、句法头/语义头的偶然分工——都源于这个混淆。详细分析和解决方案见[讨论记录-改进方向 § 路径零](讨论记录-改进方向.md)。

**Transformer = 固定语义空间（W）+ 逐层推进的通路（token 坐标）+ 路由（A）。** 通路不复合 = 每一层面对的是更新后的坐标 → 推理有深度。语义可复合 = W 矩阵的串联有确定含义 → 知识有结构。

**W 和 A 的边界不在"注意力 vs FFN"。** 常说的"注意力机制"被纵向剖开：QKV 投影矩阵属于 W（固定、存语义、可复合），softmax 输出的分数 a_{ij} 属于 A（动态、做路由、不可复合）。切分的标准是变/不变，不是功能模块。

**框架与真实推理的关系。** 此框架解释了 Transformer 内部结构的合理性，但并不声称 Transformer 在做真实推理。真实推理中，路由决策和语义理解不分离——"该注意什么"本身就是语义判断。Transformer 把两者按"可复合/不可复合"正交分离——这是干净的工程方案，但它和生物推理之间有本体论差距。框架的正交分离同时是它解释力的来源和它的边界。

## 逐一解释

### 权重 W = 语义空间

W 训练后固定。`W₂W₁` 始终有定义——权重跨层可复合。米田在 ℝᵈ 上成立——`⟨ℝᵈ|`（所有可作用在语义空间上的算子）唯一确定语义空间。这和"一个词的含义由所有可以替换/修饰/关联它的词来定义"是同构的。Anthropic 的电路分析（OV 电路、QK 电路）就是在显式读出 W 复合中编码的具体语义操作。

### Token 向量 |x_i⟩ = 路由状态

`|x_i⟩` 是路由状态向量——它是 A（路由系统）在语义空间中的当前落脚点。它随层变化，每层被 W 和残差重写。它携带的内容看起来是"语义"，但这个语义完全由 W 编码——关掉 W，|x_i⟩ 退化为无意义数值。token 标签 {i} 不构成范畴（米田不适用），因为 token 的身份不由自己决定——由它当前的 |x_i⟩ 决定，而 |x_i⟩ 的语义内容寄生在 W 上。

**架构混淆**：|x_i⟩ 在当前 Transformer 里同时做两件事——被 W 做语义变换（MLP, W_V W_O）和被 A 做路由判断（Q·K^T）。语义变换需要高维、稠密、跨样本稳定的表征；路由判断需要低维、结构化的句法位置和推理阶段信号。同一个向量的同一组维度不能同时最优地满足两者。这导致：深层路由退化（语义累积淹掉路由信号）、SAE 特征不稳定（同一概念在不同句法位置被编码在不同方向）、句法头和语义头的分工是"偶然的降维投影"而非结构性分离。

### 注意力分数 A = 路由

`a_{ij} = softmax⟨x_i|W_Qᵀ W_K|x_j⟩`。W_Q 和 W_K 属于 W（语义空间——存"问什么"和"有什么"的固定知识），它们用来计算 a_{ij}，但 a_{ij} 本身不属于 W——它每层重算、依赖当前坐标，属于 A。用 W 中存储的语义对当前 token 坐标做匹配，决定下一步信息从谁流向谁。坐标每层被 W 和残差改写 → 下一层注意力面对的是新坐标 → 路由逐层重算 → CK 不成立。

### 残差 = 路径依赖

`|x_i'⟩ = |x_i⟩ + ...`。残差保证每一步保留上一步的坐标，下一步的路由基于新坐标做出。路径依赖 = 推理深度有意义。去掉残差 → 路径依赖断裂 → 每层从零开始路由 → 深层退化为浅层。

### 激活 = 非线性

softmax 放大内积差异，ReLU/GeLU 阈值开关，LayerNorm 折叠轨线。非线性 + 路径依赖 + 局部扩张 → 推理通路可产生混沌（初值敏感）→ 相似的输入可以分叉到不同的推理路径 → 语义簇（概念的内部结构）涌现。

## 命题与论证

### 命题一：W 可复合（语义空间自足）

`(I ⊗ W₂)(I ⊗ W₁) = I ⊗ W₂W₁`。矩阵乘法封闭，结合律成立，I 为单位元。W 在复合下构成幺半群——两个语义变换串联仍是语义变换。语义空间中的向量由所有可作用其上的线性变换（等价于所有线性泛函）唯一确定——这是双对偶同构 V** ≅ V 的直接推论。■（严格证明见[完备证明 §2](完备证明.md#2-命题一w-可复合语义空间自足)）

### 命题二：A 不构成跨层范畴（通路不可复合）

```
a_{ij}^(l+1) = softmax(⟨x_i^(l+1)|W_Qᵀ W_K|x_j^(l+1)⟩)
|x_i^(l+1)⟩ = |x_i^(l)⟩ + Σ_k a_{ik}^(l) W_O W_V |x_k^(l)⟩ + MLP(|x_i^(l)⟩)
```
A^(l+1) 的输入是被残差 + W + MLP 联合改写后的坐标，不由 A^(l) 单独决定。矩阵乘积 A^(l+1)·A^(l) 存在，但不代表两步注意力的真实复合——它丢失了残差项 |x^(l)⟩ 的贡献。CK 不成立。Toy Case 已验证。（[11-ToyCase.md](11-ToyCase.md)）■（严格证明见[完备证明 §3](完备证明.md#3-命题二a-不构成跨层范畴通路不可复合)）

### 命题三：若 A 可复合，路由退化

A 全正固定 → A^L 收敛到每行皆为平稳分布 π 的秩一矩阵（Perron-Frobenius）。所有 token 获得相同的注意力上下文 → 推理路径无区分 → 语义聚类不可能。■（严格证明见[完备证明 §4](完备证明.md#4-命题三若-a-可复合路由退化)）

### 命题四：W 和 A 因果耦合

W 可跨层复合（$\text{End}(\mathbb{R}^d)$ 封闭）和 A 必须重算（坐标每层变化）不是因果关系——两者是同一 $I+G$ 残差结构的两个并行的结构性后果。$I$ 保留历史使坐标每层不同 → A 重算；$G_l$ 中的 $W$ 矩阵在 $\text{End}(\mathbb{R}^d)$ 中封闭 → W 可复合。■（修正后的证明见[完备证明 §5](完备证明.md#5-命题四w-与-a-的结构耦合)）

### 命题五：语义簇（正反馈）

相似坐标 → 相似 A → 相似更新 → 保持相似（角空间非分化 + Euclidean 有界发散）。相异则分化。正反馈。此命题是命题二和三的动力学推论——已完成定理化（角非分化下界 + 分化概要），Toy Case 同簇侧验证一致（非分化下界满足，增强效应超出理论预测），分化侧因 2 层限制未显现。■（定理化证明见[完备证明 §6](完备证明.md#6-命题五语义簇正反馈的动力学基础)）

### 命题六：嵌套米田循环（语义的层间遗传）

**单层就是一个完整的内外米田循环：**

$$x^{(\ell)} = \underbrace{x^{(\ell-1)}}_{\text{内米田骨架}} + \underbrace{\text{Attn}(x^{(\ell-1)}) + \text{FFN}(x^{(\ell-1)})}_{\text{非米田函子}}$$

输入是上一层的"外米田"（上一轮循环增强的骨架），经过非米田函子（降维 · 重路由 · 非线性阻断），输出成为下一层的"内米田"（更新的骨架）。

**L 层 = L 个循环嵌套：**

$$x^0 \xrightarrow{F_1} x^1 \xrightarrow{F_2} x^2 \xrightarrow{} \cdots \xrightarrow{} x^L \xrightarrow{W_U \cdot \text{softmax}} \text{纯米田输出（条件概率分布）}$$

$$F_\ell = I + \text{Attn}_\ell + \text{FFN}_\ell \quad \text{（非米田函子：降维·路由·阻断）}$$

**H 头 = 每层 H 条并行非米田变异。** 每头一条降维态射，并行覆盖关系空间的不同子空间。

**信息通路总数 = L × H × n**（n 为序列 token 对数）。L 决定推理步数上限，H 决定每步的关系覆盖宽度。两者相乘 = 残差流上用不完整态射逼近完备米田态射集的空间容量。

**推论一——单层无推理：** L = 1 → 非米田变异不足 → 仅有一轮重路由 → 近似固定线性变换。推理需 L ≥ 2。

**推论二——低头无宽覆盖：** H 太低 → 态射分片不够 → 每层的关系视角不完整 → 外米田输出贫瘠。宽度需要 H 下界。

**推论三——残差流不覆盖是必要条件：** 若残差流覆盖（RNN 式的 $h_t$ 替换 $h_{t-1}$）→ 早期循环的骨架被丢弃 → 嵌套深度虚增 → L 大但有效推理步数小。残差流加性保证每代骨架完整遗传。

**推论四——余米田边界条件：** 内层纯非米田 → 函子没有关系信号可投 → 外层条件概率退化为先验 → 语言模型任务无定义。因此内层的米田骨架（残差流加法、W 复合、加性保存）不是可选项——它是外米田存在的供给侧前提。详见[余米田边界条件](余米田边界条件.md)。

此命题将 W/A 正交、残差流保存、CK 不成立、语义簇统一为同一嵌套结构的必然特征——**语义不是一层的事，是 L × H 代循环的累积遗传。** ■（形式化证明见[完备证明 §7](完备证明.md#7-命题六嵌套米田循环语义的层间遗传)）

## 结论

**定位了语义空间的数学载体：权重 W 在复合下构成幺半群/范畴，语义空间中的向量由可作用其上的变换唯一确定——这与分布语义是同一结构约束的不同语言表述。**

**Transformer 的完整结构 = L × H 代嵌套循环（命题六），每代包括：**

```
内米田骨架 ──→ 非米田函子（降维·重路由·阻断）──→ 外米田骨架（= 下一代的起点）
```

- **W 是固定语义空间**（态射模板库，跨层可复合）
- **A 是每代重算的路由**（用 W 存储的模板在当前输入上做匹配，不可复合）
- **残差流是骨架的遗传载体**（只加不覆盖，保证每代骨架完整传递）
- **W/A 正交**：W 来自训练数据（训练后固定），A 来自当前输入（输入不同全变）。两者来源正交，结构共存——W 可复合（存知识）和 A 必重算（做路由）是同一 $I+G$ 结构的两个并行后果

**生效的逻辑：** L × H 条不完整态射在残差流上逐代累积遗传，以残缺拼完整——用大量降维、不对称、不可复合的"伪态射"的叠加，逼近一个在输出端够用的米田完备性。非米田变异提供推理力（抗坍缩），米田骨架保证输出合法性（抗混沌）。内层纯米田 → 坍缩为固定线性函数。内层纯非米田 → 条件概率退化为先验。Transformer 的有效域恰好在这两根边界之间（[余米田边界条件](余米田边界条件.md)）。

**|x_i⟩ 的混淆：** 当前 Transformer 中，|x_i⟩ 同时服务于 W 的语义变换和 A 的路由判断——这两件事对向量空间的需求不同（高维语义 vs 低维路由），混在同一个向量里是架构层面的欠分离。这解释了深层注意力退化、SAE 特征不稳定等现象——最简修复是将残差流分出一段独立的路由维度（d_r ≈ 64），路由信号不过 MLP，让优化自然分离两组信息。见[讨论记录-改进方向 § 路径零](讨论记录-改进方向.md)。

**框架的边界：** 此正交分离解释了 Transformer 的结构合理性，但同时也是它和真实推理之间的本体论差距所在——真实推理中路由和语义不分离。这是框架诚实性的来源。

**改进方向：** 框架的逻辑终点指向让路由成为学习对象（而非纯计算结果）——见[讨论记录-改进方向](讨论记录-改进方向.md)。

**最终定位：** 此框架排除了两个极端——Transformer 不是任意堆砌的参数盒子（每个关键组件在三组张力的解中有必要性论证），也不是语义本质的终结形态（三个米田缺口——降维、不可复合、不对称——精确标注了它离完备结构的距离）。它在两者之间的一个精确位置上：**第一代有态射结构意识但态射还不完备的语言推理机器。** 框架的价值在于给 Transformer 的技术特征在语义结构的地图中标出了一个坐标——不是终点，但坐标本身是确证的。最接近真实语义关系的结构不应是残缺态射、降维投影、因果单向——但 Transformer 用残缺拼接出了表面完备，且这个拼接的逻辑是严密可分析的。缺口在，位置在，方向在。三者同时成立。

## 可检验预测

| # | 预测 | 检验 |
|---|------|------|
| 1 | 语义簇随层增强 | 逐层 silhouette score |
| 2 | λ_max ≈ 0 | 逐层 Jacobian 谱 |
| 3 | 残差系数 ε→0 坍缩 | |x⟩ + ε·(注意力+MLP) |
| 4 | A 固定 → 退化 | 参数共享 + CK 损失 |
| 5 | 路由信号低维 | 训练线性探针：从 \|x_i⟩ 预测 a_{i*} 和输出 token。路由探针的有效秩应远低于语义探针（逐层 PCA / 探针 rank） |
| 6 | 深层路由退化先于语义退化 | 逐层 KL(a_i^(l) \|\| a_i^(l-1)) 在中间层后趋于零，但语义更新量 ‖\|x_i^(l)⟩ − \|x_i^(l-1)⟩‖ 持续非零 |
| 7 | d_head 增大不显著改善注意力 | d_head ∈ {32, 64, 128, 256, 512}，路由信号已被低维捕获→大 d_head 边际收益递减 |
| 8 | L×H 不变，L/H 偏移影响推理质量 | 固定 L×H（总态射分片数），改变 L（如 6→12→24）+ 对应反向调 H。预测存在最优 L（推理步数充足但非冗余）和最优 H（关系覆盖充分但单头不窄于阈值 d_k ≈ 32），偏离区间则 perplexity 下降 |
