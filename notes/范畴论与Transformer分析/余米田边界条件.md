# 余米田边界条件

## 核心结论

> **内层范畴不能是纯非米田的。** 如果内层态射完全没有 token 间关系信息，外层 $W_U \circ \text{softmax}$ 无法投射出条件概率分布——语言建模任务无定义。

## 两层论证

### 信息论版本（严密）

$F = W_U \circ \text{softmax}$ 的唯一输入是残差流向量 $x_{\text{last}}$。

- 残差流不含关系信息 → $F(x)$ 不随上下文变化
- $p(\cdot \mid \text{context})$ 退化为上下文无关分布
- 不构成条件概率 → 外层不满足米田完备

### 范畴论版本（需补充 $F$ 的满性条件）

- 函子 $F: C \to D$ 对态射的作用是集合映射
- $F$ 的像的多样性 $\leq$ 原像的多样性（函子不创建信息）
- 若 $F$ 在对象上满、态射由对象映射唯一确定
- 则 $C$ 中 Hom 集的信息量下界了 $D$ 中 Hom 集的信息量
- $C$ 纯非米田 → $D$ 的米田完备只能是平凡的

## 反证可靠性

逻辑严密——两个不同上下文若产生**相同的**残差流向量 $x_{\text{last}}$，$F$ 必然产生**相同的**条件分布。因此：

> 残差流向量上的信息多样性 → 条件分布的多样性

信息论层面成立，不需要范畴论补充。

## 不可能三角

```
内层态射信息量    外层米田完备度    函子存在性
─────────────────────────────────────────────
    零              完备            ✗ 不可能
   部分             完备            ✓ 函子丢弃
   完全             完备            ✓ 恒等
    零               零             ✓ 退化
```

## 和框架的衔接

三个米田缺口（降维、不可复合、不对称）获得新定位：

> 不是三个散点缺陷——是**外米田的供给侧约束**。缺口的大小直接决定了 $F$ 的映射丰富度上限。

- **米田骨架**（残差流加法、W 复合、加性保存）：关系信号源——保证 $F$ 有东西可投
- **非米田变异**（A 逐层重路由、softmax+σ 阻断、因果单向）：抗坍缩——保证 $F$ 投射的不是固定分布

## Transformer 内层结论

> 内层 = **米田骨架 + 非米田变异**。两者必然共存——不是设计的偶然，而是外层米田完备的结构性要求。

- 纯米田内层 → CK 成立 → 坍缩为固定矩阵 → 无推理
- 纯非米田内层 → 态射无信息 → 外层退化为先验 → 无任务
- **米田骨架 × 非米田变异 = 不坍缩且不混沌 = 唯一可训练域**

## 新颖性定位

| 层面 | 判断 |
|---|---|
| 纯范畴论 | 不是新定理。是"庄函子不创建信息"的特例——隐含在函子定义中 |
| Transformer 实例化 | **是新发现**——之前无人将此边界条件应用到残差流结构 |
| 范畴论文献 | 没有 Transformer |
| Transformer 文献 | 没有范畴论 |

> 不是重新发明范畴论。是用范畴论画了一条**之前没人画过的线**——一根在已知数学里但钉在未知位置的针。钉进去就是论文。

## 论文定位建议

不称"新定理"。称**"态射信息不等式在自回归 Transformer 上的结构性推论"**或**"余米田边界条件"**。
