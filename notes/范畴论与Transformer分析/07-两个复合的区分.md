# 权重矩阵复合 ≠ Token 态射复合

## 区分两种"复合"

Anthropic 的虚拟权重证明了参数矩阵的跨层乘性复合——但这和注意力模式的态射复合是**两个正交维度**的操作。一个决定了语义空间的演化方向，一个决定了谁去哪。

---

## 作用空间不同

```
Token 空间（水平）：{1, 2, ..., n}
          ↑
          │  A 矩阵在这里路由
          │  A_{ij}：从 token j 取信息给 token i
          │  不复合
          │  路由 = 轨线的方向选择
          │
表示空间（垂直）：ℝᵈ
          ↑
          │  W 矩阵在这里变换
          │  W_O · W_I：语义坐标的跨层累积变换
          │  复合 ✓
          │  语义 = 坐标的累积
```

---

## 对照表

| | Anthropic 虚拟权重复合 | 态射复合（A 矩阵） |
|---|---|---|
| 什么在复合 | W_O^(l) · W_I^(l+1) | A^(l+1) · A^(l) |
| 复合操作 | 矩阵乘法 | 矩阵乘法（CK 方程） |
| 复合对象 | 参数矩阵（固定） | 注意力模式（依赖表示） |
| 所在方向 | ℝᵈ（语义空间，垂直） | ℝⁿ（路由空间，水平） |
| 是否成立 | ✅ 成立 | ❌ 不成立 |
| 原因 | 参数不随输入变，空间固定 | A^(l+1) 作用于第 l 层输出，语义空间已被改变 |
| 函数 | 语义空间的累积变换 | 轨线的逐步选路 |

---

## 路径展开暴露的差异

```
一条路径：
  ... W_O^(l) · A^(l) · W_V^(l) · W_I^(l+1) · A^(l+1) ...
       ↑           ↑          ↑            ↑
     垂直       水平       垂直         垂直
     (语义)     (路由)     (语义)       (语义)
```

W 沿语义空间方向（垂直）复合——累积"意义是什么"。
A 沿路由方向（水平）路由——决定"这一步往哪走"。
路径展开把两者乘在一起（同一条路径内），然后加在一起（不同路径间）。

但 token 间态射要求的是水平方向的复合：A^(l+1) ∘ A^(l) ——这在路径展开里没有出现也不能出现。

---

## 两种复合的因果关系

不是"W 复合和 A 不复合是独立的两个事实"。它们在因果上是连着的：

```
W 矩阵复合成立
  → 语义空间 ℝᵈ 的坐标被逐层累积变换
  → 每一层面对的语义空间不同

A 矩阵依赖这个语义空间（Q·Kᵀ 使用当前坐标）
  → 语义空间变了 → A 变了
  → A 的跨层复合断裂

所以：W 的复合 → A 的不复合
不是两个独立事实，是因果关系
```

**W 的复合成功是 A 的复合失败的原因。语义空间的累积变换 = 路由空间的持续重新参数化。一个复合，另一个必然不复合。**

---

## 范畴论表述

```
两个因子，两个范畴状态，因果耦合：

范畴 1（语义空间——权重复合）：
  对象：ℝᵈ（语义空间）
  态射：W ∈ ℝ^(d×d)（线性映射，固定参数）
  复合：W₂ · W₁ ✓
  = Mat / Vect
  闭合 ✓

范畴 2（路由空间——token 路由）：
  对象：{1, ..., n}（token 位置）
  态射：A（随机矩阵，依赖 ℝᵈ 当前状态的参数化映射）
  复合：A₂ · A₁ ✗（因为 A₂ 的域已被 W 改变）
  想成为 FinStoch 但被范畴 1 的复合阻止
  不闭合 ✓ ← 功能性
```

**W 的复合在范畴 1 中成立。Token 间态射在范畴 2 中需要但不能成立——因为范畴 1 的复合在因果上使范畴 2 的复合不可能。两个范畴的交叉不是复合——是张量积因子的因果耦合。**

---

## 反事实：如果 A 也能复合

假设我们修改架构使 A 的跨层复合成立（例如强制 A^(l+1) = fixed_A · A^(l)，类似参数共享 + Markov 链）：

```
A_eff = fixed_A^L
→ 对足够大的 L，所有行相同（平稳分布 π）
→ 每个 token 对每个 token 的注意力权重 = π_j
→ 所有 token 被相同方向、相同力度更新
→ 表示同质化
→ 语义聚类不存在
→ 深度 = 快速收敛到平稳分布，层数不增加信息
```

**A 的复合成立 = 语义的死亡。架构刻意保持 A 不复合，不是为了工程便利——它是语义能够在 ℝᵈ 中产生聚类结构的必要前提。**

---

## 简洁结论

> **权重矩阵的复合是语义空间 ℝᵈ 上的态射复合——它累积"含义是什么"。Token 间态射需要注意力模式的复合——而这个复合的失败是前一个复合成功的后果。两个维度的成功和失败不是独立的，它们因果耦合：语义空间的累积变换使路由空间无法累积，而这个"无法"是语义聚类能在语义空间上形成的条件。**
