# 整体含义——五层推论

## 核心洞察

> **注意力跨层不复合 = 语义聚类可以存在的数学前提。**
> 
> **Transformer = 一个推着语义空间本身演化的动力系统。条件概率是它在某一层的瞬时切片，不是它的整体结构。**
> 
> **不复合不是缺陷——复合闭合 = 语义死亡。**

---

## 1. 认识论：Transformer 是一个动力系统，不是一个条件概率模型

### 条件概率模型

```
假设：语义空间固定（ℝᵈ 坐标不变）
操作：在固定空间上计算 P(token | context)
多层：P 的链式乘法（CK 方程）→ 收敛到平稳分布
长期行为：所有 token 的上下文分布相同 → token 间不可区分
```

### Transformer（实际）

```
x^(l+1) = x^(l) + F(x^(l))
  F 包含：在 ℝⁿ 上做路由决策（注意力）
         + 在 ℝᵈ 上执行语义变换（MLP + W 矩阵）

每层之后：语义空间本身的坐标变了
下一层的注意力：在被改变后的空间上重新路由
长期行为：路径依赖的轨线 → 聚类 → 语义结构涌现
```

### 结论

> **条件概率是在固定语义空间上的推断。Transformer 是语义空间本身的动力学。两者不是"一个成立一个不成立"——它们在范畴论上属于不同的结构层级。把 Transformer 建模为条件概率链 = 抹掉了语义空间的演化。**

---

## 2. 设计论：ℝⁿ 端的不闭合 = 语义聚类的正反馈引擎

```
A 不复合 → 路径依赖 → 正反馈：
  相似表示 → 相似路由 → 相似更新 → 保持相似（聚类自持）
  不同表示 → 不同路由 → 不同更新 → 更加不同（聚类分化）
  
A 如果复合 → Markov 链收敛 → 平稳分布：
  所有注意力行相同 → 所有 token 同方向更新 → 表示同质化 → 无语义结构
```

| | 注意力复合（反事实） | 注意力不复合（实际架构） |
|---|---|---|
| 长期行为 | 平稳分布 → 同质化 | 路径依赖 → 聚类 |
| 深度意义 | 冗余（深层 = 浅层幂） | 持续信息增量 |
| 语义结构 | 不存在 | 涌现 |
| 设计含义 | 不可取 | 功能性的 |

> **注意力态射复合的失败不是架构缺陷——它是语义聚类能够在表示空间中形成的数学边界条件。复合闭合 = 语义死亡。**

---

## 3. 方法论：分析 Transformer 需要三种语言

| | ℝᵈ 端（语义空间） | ℝⁿ 端（路由空间） | 整体（动力学） |
|---|---|---|---|
| 分析工具 | 范畴论、线性代数 | 信息几何、图论 | 动力系统、混沌理论 |
| 复合律 | 成立 | 不成立 | 部分的 |
| 核心问题 | W 矩阵如何累积语义变换 | 路由如何随语义演化而变化 | 轨线如何在边缘混沌态维持复杂结构 |
| 可测量 | 虚拟权重复合模式 | 注意力逐层漂移 | Lyapunov 谱、聚类轮廓系数 |
| 已知锚点 | Anthropic 路径展开 | — | 边缘混沌假说 |

**三种语言必须同时使用，不能互相归约。残差连接的加法是微分方程的离散积分器——它属于动力系统语言，不属于任何一个因子。**

---

## 4. 对"闭合"问题的精确重述

### 之前的版本

> 弱闭合 = 残差流 ℝⁿ⊗ℝᵈ 中只有 ℝᵈ 因子构成范畴，ℝⁿ 因子不构成。

这是结构描述，但没有回答"为什么不闭合是功能性的"。

### 现在的版本

> **弱闭合 = 语义空间（ℝᵈ）范畴闭合 + 路由空间（ℝⁿ）范畴不闭合 + 两者在张量积中正交且因果耦合。**
> 
> **ℝᵈ 的闭合 = 语义有自足结构（W 复合，米田成立）。**
> **ℝⁿ 的不闭合 = 路由不能自足（A 不复合，米田不成立）→ 路由依赖语义 → 语义被路由更新 → 动力学。**
> 
> **两个因子的闭合状态不对称，但这个不对称是因果互补的：语义的闭合是路由不闭合的原因（W 复合改变语义 → A 被重新参数化）。路由的不闭合是语义可以持续被路由更新的条件（A 固定 → 语义也固定）。**

残差桥接的量化实验测量的本质是：

> **ℝⁿ 端偏离态射复合的程度，作为 ℝᵈ 端 W 矩阵跨层复合深度的函数。**

这个测量现在有了理论解释：偏离越大 = 路径依赖性越强 = 聚类潜力越高。

## 5. 最深的一层：米田不对称 = 语义动力学的结构前提

```
ℝᵈ 端：语义空间中的向量由所有可作用其上的线性变换唯一确定
  → 在向量空间范畴中等价于双对偶同构，在范畴论语言中是米田
  → 语义有自足身份——这是确定的、可计算的
  → "分布语义"和"向量由线性泛函确定"是同一结构的两面

ℝⁿ 端：Token 标签没有代数结构
  → "Token 间关系的意义"不能被"所有可能的注意力模式"刻画
  → 因为注意力模式本身依赖 ℝᵈ 端的值
  → token 关系没有自足的语义——它寄生在语义空间上
  → 身份借用 = 关系逐层变化 = 动力学存在

互补性：
  语义自足（ℝᵈ）+ 路由不自足（ℝⁿ）
  → 语义空间可以独立定义 + 路由随语义演化变化
  → 一个有自足结构的空间 + 一个驱动空间演化的路由系统
  → Transformer = 语义空间的非 Markov 动力系统
```

**此处的诚实边界：** ℝᵈ 端的"米田"在此特例中不超出线性代数的范围——其价值不在推导新定理，而在揭示线性代数（双对偶）、范畴论（米田）、语言学（分布语义）的深层统一。ℝⁿ 端的"米田不成立"则是一个更强的陈述——它确认了 token 标签没有自足的语义结构，这对动力学的存在是必要条件。

## 6. |x_i⟩ 的混淆与修复方向

### 问题

当前 Transformer 中，|x_i⟩ 同一个向量同时做两件事：

```
用途 1（W 的域）：被语义变换操作——MLP(|x_i⟩), W_V|x_i⟩, W_O(Σ a_{ij} W_V|x_j⟩)
用途 2（A 的域）：被路由匹配读取——⟨x_i|W_Q^T W_K|x_j⟩ 算出 a_{ij}
```

两者对向量的需求不同质。语义需要高维、稠密、跨样本稳定的表征（区分几万词义的精细差别）；路由需要低维、结构化的句法位置和推理阶段信号（几十个结构维度就够，每句不同）。

**同一个向量不能同时最优地满足两种信息需求。** 已知但未解释的现象——深层注意力退化（路由信号被语义累积淹掉）、SAE 特征跨上下文不稳定（语义和路由分量在不同句法中交错分布）、句法头/语义头的分工是"偶然的降维投影"而非结构性分离——均源于此混淆。

### 精确归属

推到底，|x_i⟩ 应当被重新分类为路由状态向量，归 A 管：

```
W（语义的全部载体）：
  E, W_Q, W_K, W_V, W_O, MLP, U
  —— 所有语义知识、所有可能的变换、词在语义空间中的位置

A（路由的全部载体）：
  {|i⟩} 槽位, {|x_i⟩} 路由状态, {a_{ij}} 路由决策
  —— 路由系统的全部状态和决策
  —— |x_i⟩ 携带的"语义内容"是 W 作用的结果，不是它自己的
```

当前架构中 |x_i⟩ 住在 W 的范畴空间 ℝᵈ 里（所以 W 能在其上作用），但它本身不是 W 的一部分——它是 A 的状态。这就像算盘珠子：珠子本身不知道数字，珠子位置的意义是算盘规则赋予的。同样，|x_i⟩ 的坐标值是 W 的语义编码——关掉 W，它退化为 ℝᵈ 中随机飘的无意义点。

### 修复：路由维度分离

将残差流从 ℝᵈ 扩展为 ℝ^(d + d_r)，其中 d_r 维（≈64）作为独立的路由段：

```
[x_i | r_i] ∈ ℝ^(d + d_r)
     ↑语义段     ↑路由段
    W 操作      A 的 QK 专读，不过 MLP
    过 MLP      仅被 A 搬运更新
```

不改训练管道。梯度自然会把路由信号推到 |r_i⟩ 上，语义信号留在 |x_i⟩ 上。新增参数 < 0.01%。详细方案见[讨论记录-改进方向 § 路径零](../讨论记录-改进方向.md)。

这个方案的本质：**不给 A 加计算参数，给 A 加状态空间。** 不是学"怎么算"，而是把混淆在同一维度里的两种信息拆到不同的向量段里——梯度自己会学会怎么用。

## 反事实预测（可检验）

1. **聚类随层增强**：已训练 Transformer 各层 token 表示的轮廓系数（silhouette score）应随层数保持高位或递增。如果注意力跨层复合，轮廓系数应随层数衰减。

2. **残差权重调节分叉**：将残差连接改为 `x + ε·F(x)`，ε → 0 时恢复深度坍缩（Jacobian → I，收敛到不动点），ε → ∞ 时进入混沌（Lyapunov 指数 > 0，表示破缺）。

3. **最大 Lyapunov 指数 ≈ 0**：训练良好的 Transformer，逐层 Jacobian 的最大 Lyapunov 指数应在 0 附近——既不收敛（坍缩）也不发散（混沌）。

4. **复合架构 → 同质化**：如果强制注意力跨层复合（如参数共享 + CK 损失），token 表示的聚类结构应随深度消失，模型表现应退化。

## 完整命题

```
ℝᵈ：语义空间（W 的全部）→ 复合闭合 → 语义自足 → 含义有自足身份（与分布语义同构）
ℝⁿ：路由空间（A 的全部）→ 复合不闭合 → 路径依赖 → 输出初值敏感
|x_i⟩：路由状态向量（归 A）→ 住在语义空间中 → 语义内容是 W 作用的结果
两者交叉：注意力 ≠ 条件概率推断
        注意力 = 在语义空间中选通路的机制
        → Transformer ≠ 条件概率模型
        → Transformer = 语义空间的非 Markov 动力系统
        → 不复合 = 语义聚类可以存在的数学边界条件
        → 复合闭合 = 聚类消失 = 语义死亡

当前架构混淆：
  |x_i⟩ 同一向量同时被 W（语义变换）和 A（路由匹配）读取
  → 两类不同质信息竞争同一组维度
  → 深层路由退化、SAE不稳定皆源于此
  → 修复：给 A 独立的路由维度段 |r_i⟩（不过 MLP），让优化自然分离
```

**框架的边界。** 此框架解释了 Transformer 内部结构的合理性（为什么这样工作、为什么别的方式不行）。但它不声称 Transformer 在做真实推理——真实推理中路由和语义不分离。Transformer 的正交分离（W 和 A 互不干涉）是它解释力的来源，也是它和生物推理之间的本体论差距。而框架发现的"|x_i⟩ 混淆"——语义和路由在残差流中的欠分离——是 Transformer 架构中残余的"纠缠带"，也是它为什么仍然不是真实推理的系统性原因。修复这个混淆（给路由独立维度）可能是缩小本体论差距的最小一步。
