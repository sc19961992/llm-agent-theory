# Toy Case：手算验证框架

3 token，2 维，无 MLP，权重 = I（简化到极致）。

---

## 张量积框架

```
残差流：|Ψ⟩ = Σ_i |i⟩ ⊗ |x_i⟩

|i⟩ ∈ ℝ³   位置 ket（token 槽位）
|x_i⟩ ∈ ℝ² 语义 ket（token 向量）

一层 = 注意力混位置 + 残差保留原 ket：

(A ⊗ I) |Ψ⟩ = Σ_i |i⟩ ⊗ (Σ_j a_{ij} |x_j⟩)       ← 注意力：碰位置，混语义
(I ⊗ W) |Ψ⟩ = Σ_i |i⟩ ⊗ W|x_i⟩                 ← 权重：不碰位置，变语义
(A ⊗ I)(I ⊗ W) = A ⊗ W = (I ⊗ W)(A ⊗ I)       ← 对易

本例：W_Q = W_K = W_V = W_O = I₂（权重不做变换，专注看注意力）
     MLP = 0（无激活，专注看注意力复合）
```

---

## 符号表

| 符号 | 含义 | 本例取值 |
|------|------|---------|
| \\|i⟩ | 位置 ket（∈ ℝ³，第 i 个 token 槽位） | — |
| \\|x_i⟩ | token 坐标（∈ ℝ²，语义空间中的当前位置） | 见下 |
| ⟨x_i\\| | 语义 ket 的对偶（行向量 = 转置） | — |
| ⟨x_i\\|x_j⟩ | 内积 = 注意力原始分数 | — |
| s_{ij} | 注意力分数（softmax 入参） | s_{ij} = ⟨x_i\\|x_j⟩ / √2 |
| a_{ij} | 注意力权重（softmax 输出） | a_{i*} = softmax(s_{i*}) |
| A^(l) | 第 l 层注意力矩阵（3×3） | — |
| A ⊗ I | 注意力作用在两体态上：碰位置 | — |
| I ⊗ W | 权重作用在两体态上：碰语义 | 本例 W = I₂ |
| \\|x_i'⟩ | 更新后的语义 ket（下一层输入） | \\|x_i⟩ + Σ_j a_{ij}\\|x_j⟩ |
| √2 | 缩放因子 = √d_head | — |
| softmax(z)_j | exp(z_j) / Σ_k exp(z_k) | — |
| cos(u,v) | 余弦相似度 | ⟨u\\|v⟩ / (‖u‖·‖v‖) |
| CK | Chapman-Kolmogorov 方程 | A^(l+1)·A^(l) 应等于两步转移 |
| π | Markov 链平稳分布 | A^∞ 的任意一行 |

## 初始

```
|x₁⟩ = [2.0, 0.0]ᵀ    "dog"
|x₂⟩ = [1.8, 0.2]ᵀ    "wolf"（语义近 dog）
|x₃⟩ = [0.0, 2.0]ᵀ    "car"（语义远）

即：|Ψ^(0)⟩ = |1⟩⊗[2.0,0.0]ᵀ + |2⟩⊗[1.8,0.2]ᵀ + |3⟩⊗[0.0,2.0]ᵀ

前向：|Ψ^(l+1)⟩ = |Ψ^(l)⟩ + (A^(l) ⊗ I) |Ψ^(l)⟩
     a_{ij}^(l) = softmax(⟨x_i^(l)|x_j^(l)⟩ / √2)_j
```

---

## 第一层：注意力

### 注意力分数

```
s_{ij} = ⟨x_i|x_j⟩：

          j=1    j=2    j=3
    i=1:  4.00   3.60   0.00
    i=2:  3.60   3.28   0.40
    i=3:  0.00   0.40   4.00
```

除以 √2 ≈ 1.414，softmax：

```
row 1:  softmax(2.828, 2.545, 0.000)
      = [0.552, 0.416, 0.033]

row 2:  softmax(2.545, 2.319, 0.283)
      = [0.526, 0.419, 0.055]

row 3:  softmax(0.000, 0.283, 2.828)
      = [0.052, 0.069, 0.879]
```

**dog 和 wolf 的注意力模式高度相似**——都主要从 1、2 取信息，car 则只看自己。

### 新 ket

```
|x₁'⟩ = |x₁⟩ + 0.552|x₁⟩ + 0.416|x₂⟩ + 0.033|x₃⟩
      = [2,0]ᵀ + [1.104,0]ᵀ + [0.749,0.083]ᵀ + [0,0.066]ᵀ
      = [3.853, 0.149]ᵀ

|x₂'⟩ = [1.8,0.2]ᵀ + 0.526[2,0]ᵀ + 0.419[1.8,0.2]ᵀ + 0.055[0,2]ᵀ
      = [3.606, 0.394]ᵀ

|x₃'⟩ = [0,2]ᵀ + 0.052[2,0]ᵀ + 0.069[1.8,0.2]ᵀ + 0.879[0,2]ᵀ
      = [0.228, 3.772]ᵀ
```

### 余弦相似度

```
                层 0    层 1
cos(dog,wolf)   0.994   0.997    ← 更近了（聚类自持）
cos(dog,car)    0.000   0.099    ← 仍远
cos(wolf,car)   0.110   0.168    ← 仍远
```

**正反馈**：相似 ket → 相似注意力（a_{i*} 相近）→ 相似更新 → 仍相似。跨组永远远。

## 第二层：CK 不成立

注意力的 a_{ij} 逐层重算——因为残差（|x⟩ 项保留）和注意力输出（Σ_j a_{ij}|x_j⟩）联合改变了 |x⟩，下一层注意力从被改写后的 |x'⟩ 出发。

### 新注意力分数

```
⟨x₁'|x₁'⟩ = 3.853² + 0.149² = 14.869
⟨x₁'|x₂'⟩ = 3.853·3.606 + 0.149·0.394 = 13.953
⟨x₁'|x₃'⟩ = 3.853·0.228 + 0.149·3.772 = 1.440

⟨x₂'|x₂'⟩ = 3.606² + 0.394² = 13.158
⟨x₂'|x₃'⟩ = 3.606·0.228 + 0.394·3.772 = 2.308

⟨x₃'|x₃'⟩ = 0.228² + 3.772² = 14.280
```

除以 √2，softmax：

```
row 1: s = (10.514, 9.866, 1.018)  →  [0.656, 0.344, 0.000]
row 2: s = (9.866, 9.303, 1.632)  →  [0.637, 0.363, 0.000]
row 3: s = (1.018, 1.632, 10.098)  →  [0.000, 0.000, 1.000]
```

### CK 检验

CK 声称 `A^(2)·A^(1)` 代表两步注意力的复合。

```
A^(1) = [[0.552, 0.416, 0.033],
         [0.526, 0.419, 0.055],
         [0.052, 0.069, 0.879]]

A^(2) = [[0.656, 0.344, 0.000],
         [0.637, 0.363, 0.000],
         [0.000, 0.000, 1.000]]

A^(2)·A^(1) = [[0.543, 0.417, 0.041],
                [0.543, 0.417, 0.041],   ← 前两行完全相同！
                [0.052, 0.069, 0.879]]
```

矩阵乘积存在。但它**不代表两步注意力的复合**：

1. A^(2) 不是从 A^(1) 的矩阵输出生成的——它是从 `|x'⟩ = |x⟩ + Σ_j a_{ij}|x_j⟩` 重算的。A^(2)·A^(1) 漏掉了残差项 |x⟩。
2. 乘积的前两行完全相同——但实际 A^(2) 各行仍有差异（0.656 vs 0.637）——乘积把个性化的注意力抹平了。
3. CK 需要 Markov 性：`P(X₂) = ∫ P(X₂|X₁)P(X₁)dX₁`，这里 X₂ 不仅依赖 P(X₁|X₀)（即 A^(1)），还依赖 X₀ 本身（残差）。

**CK 不成立。注意力逐层重算——它就是"在当前 ket 配置下选路"。**

---

## 反事实：固定注意力

固定 A = A^(1) 不变（即 a_{ij} 不依赖 |x⟩——相当于米田适用于 token 标签）：

```
A^10 ≈ [[0.378, 0.349, 0.273],
        [0.378, 0.349, 0.273],   ← 所有行相同
        [0.378, 0.349, 0.273]]

A^∞ = 全部行 = π ≈ [0.378, 0.349, 0.273]
```

所有 ket 获得**完全相同的注意力上下文**：`Σ_j π_j|x_j⟩` 对 i 无差异。注意力的个性化消失。残差 |x_i⟩ 仍保留部分差异，但路由不再为不同 token 做不同的事。

## 对照

| | 实际（注意力逐层重算） | 反事实（注意力固定可复合） |
|---|---|---|
| 层 1 dog,wolf 的 a_{i*} | [0.552,0.416,0.033] vs [0.526,0.419,0.055] | 同左 |
| 层 2 dog,wolf 的 a_{i*} | [0.656,0.344,0.000] vs [0.637,0.363,0.000] ← 仍有差异 | 全相同 ← 差异消失 |
| cos(dog,wolf) | 0.994 → 0.997（增强） | 被平均化（趋近） |
| 注意力（路由） | 个性化 | 全局统一 |
