# 米田不对称性——Token 不是对象

## 核心命题

> **在 Transformer 内部，存在一个形而上学层级：特征空间有独立的存在和自足的语义身份——可由所有指向它的映射的总体来定义。Token 位置没有——它的"身份"总是从特征空间的当前状态借来的。这个不对称不是缺陷——它是语义动力学的前提。**

---

## ℝᵈ 端：米田精确成立

```
对象：ℝᵈ（语义空间）

所有指向 ℝᵈ 的态射：
  hom(-, ℝᵈ) = { M: V → ℝᵈ | V ∈ Ob(Mat) }

包括：
  - W_I: ℝᵈ → ℝᵈ（嵌入投影）
  - W_O: ℝ^(d_head) → ℝᵈ（输出投影）
  - W_O · W_V: ℝ^(d_head) → ℝᵈ
  - W_O^(l) · W_I^(l+1): ℝᵈ → ℝᵈ（虚拟权重）
  - 路径展开中的任意 W 乘积链
  - 所有可能的权重矩阵组合

米田引理：hom(-, ℝᵈ) 的结构完全确定 ℝᵈ。
```

**语义空间的身份 = 所有可以写入它的线性映射的集合 + 这些映射的复合关系。成立。**

---

## ℝⁿ 端：问题本身无法提出

要说出"token i 的语义身份由指向它的态射确定"，需要：

```
(a) token i 是一个对象           ← 需要范畴
(b) 存在 hom(-, i) 作为集合       ← 需要 hom 集
(c) hom(-, i) 唯一确定 i          ← 需要米田
```

实际情况：

```
(a) token 位置 {i} 在 ℝⁿ 中没有范畴结构
(b) A_{ji} 不是态射，是态射的分量
(c) 即使把注意力权重视为"指向 i 的映射"，
    这些权重依赖 ℝᵈ 端的当前值——
    hom(-, i) 本身不是固定的，不是 i 的性质，
    而是 (i, 当前语义空间状态) 的联合函数
```

**Token 的身份不能由"指向它的注意力"确定——因为注意力本身依赖语义空间的状态。Token 没有自足的语义身份。**

---

## 不对称性总结

```
ℝᵈ 端的对象（语义空间）：
  身份自足 ✓
  由 hom(-, ℝᵈ) 确定 ✓
  不依赖 ℝⁿ 端的当前值 ✓
  米田成立 ✓

ℝⁿ 端的"对象"（token 位置）：
  身份不自足 ✗
  不能被 hom 确定（无 hom，有 hom 也依赖 ℝᵈ）
  完全依赖 ℝᵈ 端的当前表示
  米田不适用 ✗
```

---

## 但这个不对称是功能性的——不是缺陷

如果 token 有自足身份（米田在 ℝⁿ 端成立），那意味着 token 间的关系 a_{ij} 是固定的——不依赖 ℝᵈ 端状态。但 a_{ij} 固定 = 注意力不随语义演化 = Markov 复合成立 = 平稳分布同质化 = 语义聚类消失。

**Token 向 ℝᵈ 借用身份 = a_{ij} 随语义空间变化 = 正反馈环路 = 语义聚类涌现。**

> **米田在 ℝⁿ 端的失败 = 语义结构在 ℝᵈ 端的存在条件。**
> 
> **如果 token 有自足身份，语义空间就没有动力学。**

---

## 不对称性的因果方向

```
ℝᵈ 端米田成立 → 语义空间有自足结构
  → W 矩阵可以复合 → 语义空间随层累积变换

ℝⁿ 端米田不成立 → token 身份依赖 ℝᵈ
  → A 矩阵不能复合 → 路由随层重新参数化

因果关系：
  ℝᵈ 的米田（语义自足）+ ℝⁿ 的米田失败（身份借用）
  → 一个自足的语义空间 + 一个寄生在它上面的路由系统
  → 语义空间可以演化（W 复合）
  → 路由随演化重新决策（A 不复合）
  → 正反馈 → 聚类 → 语义结构涌现
```

---

## 更深层含义

> **"一个对象的语义身份由所有指向它的态射的集合所唯一确定"——这仍然是真理。只是 Transformer 里只有一个对象满足作为"对象"的条件：语义空间 ℝᵈ。**
> 
> **但这不是设计的局限——这是设计的功能。如果 token 也是对象，语义空间就不会演化。Token 的非对象性是语义动力学的数学前提。**

**注意力矩阵 A 的条目 a_{ij} 不是 token j 对 token i 的映射——它们是被语义空间参数化后的路由决策。它们寄生在 ℝᵈ 的结构上——而正是因为这种寄生关系，它们能驱动语义空间本身的演化。**

米田引理在 Transformer 中**局部成立但不对称**——在 ℝᵈ 因子上精确成立，在 ℝⁿ 因子上不适用。**这个不对称性不是理论选择，不是架构偶然，它是 Transformer 能够产生语义结构的核心数学条件。**
