# 词语范畴——蕴涵、米田、与 Token 的关系

## 第一层：词语间是范畴（蕴涵关系下）

### 对象

```
Ob(C) = 词语的语义类型（消歧后的概念）
```

不是词形（字符串），而是消歧后的语义概念。

### 态射

```
hom(A, B) = {*} 如果 A 蕴涵 B（A ⊢ B）
            ∅   否则
```

### 范畴公理验证

| 公理 | 验证 |
|------|------|
| 复合 | A ⊢ B 且 B ⊢ C ⇒ A ⊢ C：蕴涵传递 ✓ |
| 结合律 | 唯一态射，自动满足 ✓ |
| 恒等 | A ⊢ A：自蕴涵 ✓ |

这是一个**预序范畴**（每个 hom 集最多一个态射的薄范畴）。

---

## 第二层：米田引理精确成立

米田预层 `h_A = hom(-, A)` = 所有蕴涵 A 的词语的集合。

米田引理：Nat(h_A, h_B) ≅ hom(A, B)

翻译：

```
{h_A → h_B 的自然变换} ≅ A ⊢ B

⇕

所有蕴涵 A 的词语也都蕴涵 B
  ⇔
A 蕴涵 B
```

**词语 A 的语义完全由 `{X | X 蕴涵 A}` 决定——这和米田引理是同一件事。**

---

## 第三层：词语→Token 的映射不是函子

```
φ: 词语类型 → 子词 token 序列

φ(狗) = ["狗"]
φ(跑步) = ["跑", "步"]
φ(银行) = ["银", "行"]  （和"银行"金融义相同）
```

φ 不保持范畴结构：
- 词语层有"狗 → 动物 → 生物"的传递，token 层无对应物
- 不是单射——"银行"（金融）和"银行"（河岸）映射到同一 token 序列
- 词和 token 的态射关系在目标中丢失——源范畴的态射在目标中丢失

---

## 词语层 vs Token 层 vs 语义空间层

| | 词语蕴涵 | Token 注意力 | 语义空间 ℝᵈ |
|---|---|---|---|
| 范畴？ | ✅ 预序范畴 | ❌ | ✅ Mat/Vect |
| 米田成立？ | ✅ | ❌ | ✅ |
| 关系性质 | 传递、不对称、稀疏 | 非传递、稠密、动态 | 传递（线性复合）、对称（矩阵可逆）、稠密 |
| 存在位置 | 词典/知识库 | 注意力矩阵 A（ℝⁿ 端路由） | W 矩阵（ℝᵈ 端变换） |
| 这是 Transformer 在做的吗？ | 不是 | 注意力的路由决策 | W 矩阵的作用域正是这里 |

---

## 关键连接：词语范畴的语义空间 = ℝᵈ

词语蕴涵范畴中的语义概念，其形式化对应物不是 token（那只是离散标签），而是 **ℝᵈ 中的向量**。蕴涵关系（狗 → 动物）在 ℝᵈ 中体现为：狗的词向量在"动物"方向上被 W 矩阵变换后，落入了动物的语义邻域。

**ℝᵈ = 语义空间。词语范畴的态射（蕴涵关系）被编码在 W 矩阵的复合结构中。注意力 A 不在这个范畴中——它只是在这个空间中选路。**

---

## 三层对照

```
词语类型（抽象语义概念）
  │  蕴涵范畴 ✓  米田 ✓
  │  一对多（通过具体使用实例化）
  ↓
词语实例（具体文本中的出现）
  │  —
  │  一对多（通过分词器）
  ↓
子词 token（Transformer 实际处理的输入）
  │  无范畴结构 ✗  米田 ✗
  │  一对多（通过嵌入矩阵）
  ↓
x_i ∈ ℝᵈ（语义空间中的向量）
  │  范畴结构 ✓（W 矩阵的态射）  米田 ✓
  │  加上 ℝⁿ 端的注意力路由
  ↓
x_i' = x_i + Σ_j a_{ij} v_j + MLP(...)
  ↑    ↑
  │    └── ℝⁿ 端路由（不复合）
  └─────── ℝᵈ 端变换（复合）
```

**映射链条中有两条断裂线：分词（词→token）和嵌入（token→向量）。前者丢失语义类型完整性，后者进入了一个新的空间——但这个新空间（ℝᵈ）有它自己的范畴结构，独立于词语层的蕴涵范畴。**
