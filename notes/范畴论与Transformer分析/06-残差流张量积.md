# 残差流张量积——ℝⁿ ⊗ ℝᵈ 的代数结构与动力学

## 残差流的数学形式

```
X^(l) ∈ ℝⁿ ⊗ ℝᵈ

n×d 矩阵，每行是一个 token 的 d 维表示：
X^(l) = [x₁^(l), x₂^(l), ..., x_n^(l)]ᵀ
```

**ℝⁿ**（token 空间）和 **ℝᵈ**（特征空间）是同一个张量积的两个独立因子。完全正交。

---

## 两个因子：两个正交角色

```
ℝⁿ ⊗ ℝᵈ

  │        │
  │        └── ℝᵈ：语义空间
  │             W: ℝᵈ → ℝᵈ（权重矩阵——固定态射）
  │             范畴（Mat）在这里
  │             复合 ✓  米田 ✓
  │             = 含义的坐标
  │
  └── ℝⁿ：路由空间
       A: ℝⁿ → ℝⁿ（注意力矩阵——参数化路由，依赖 ℝᵈ 状态）
       不是范畴
       复合 ✗  米田 ✗
       = 通路的选择
```

注意力 softmax 在 ℝⁿ 方向行归一化——决定"往哪走"。
权重线性映射在 ℝᵈ 方向变换——决定"那个方向是什么"。
**两个方向的操作互不干涉，但互相依赖：A 依赖 ℝᵈ 当前状态，W 固定不变。**

---

## 为什么 ℝᵈ 端复合成立，ℝⁿ 端不成立

### ℝᵈ 端（语义空间，复合成立）

```
W 矩阵：ℝᵈ → ℝᵈ
   对象 ℝᵈ 固定（维度不随层变化）
   W₂ · W₁ 始终定义良好
   每一层都在同一个语义空间中操作
   复合 = 对语义空间的累积变换
```

### ℝⁿ 端（路由空间，复合不成立）

```
A 矩阵：ℝⁿ → ℝⁿ
   对象 ℝⁿ 固定（位置数不变）
   但 A 的矩阵条目 A_{ij} = f(x_i, x_j) 依赖 x ∈ ℝⁿ⊗ℝᵈ
   而 x 在被 ℝᵈ 方向变换——W 的操作改变
   所以 A^(l+1) 的输入在 ℝᵈ 端已经被 A^(l) 输出改变了
   A^(l+1) 不是纯 ℝⁿ → ℝⁿ 的固定态射
   它是 ℝⁿ⊗ℝᵈ → ℝⁿ 的参数化映射
   
复合断裂不是因为 ℝⁿ 变了（它没变），
而是因为 A 是 ℝⁿ⊗ℝᵈ 的函数，而 ℝᵈ 端被逐层改变了。
```

---

## 动力系统视角：为什么加法是函数性选择

残差连接 `x^(l+1) = x^(l) + F(x^(l))` 是一个**离散 Euler 积分器**。

### 如果残差用乘法

`x^(l+1) = x^(l) ⊙ F(x^(l))` 将导致：
- 若 |F| < 1 → 轨线指数衰减到 0 → 深度坍缩
- 若 |F| > 1 → 轨线指数发散 → 表示爆炸
- 对于有界非线性 σ，几乎所有轨线趋向 0

**乘法残差不能维持长期的复杂轨线。**

### 加法残差的动力学意义

`x^(l+1) = x^(l) + F(x^(l))`：
- 恒等项 I = 轨线保持当前位置的记忆
- F 项 = 在当前位置上做非线性修正
- Jacobian = I + ∂F/∂x：雅可比的特征值决定轨线行为
- 最大 Lyapunov 指数 ≈ 0 → **边缘混沌** → 轨线既不收敛也不发散

**加法是唯一能长期维持边缘混沌的累积方式 = 唯一能让深度持续产生复杂语义结构的积分器。**

## 残差空间的分解

```
残差流 = ℝⁿ ⊗ ℝᵈ

ℝᵈ 因子（语义空间）：
  W 矩阵在这里
  态射复合成立 ✓
  范畴（Mat）完整闭合
  米田成立 ✓
  语义空间有自足结构
  复合子空间

ℝⁿ 因子（路由空间）：
  A 矩阵在这里
  态射复合不成立 ✓  ← 不是缺陷
  被 ℝᵈ 端操作逐层重新参数化
  米田不成立 ✓      ← 不是缺陷
  不复合子空间
  路径依赖 = 语义聚类的正反馈引擎
```

**同一空间，正交分解，一方闭合（语义），一方不闭合（路由）。两者的交叉 = 语义空间上有复杂轨线的动力系统。**

## 路径展开的双角色

Anthropic 的路径展开反映这个分解：

```
Σ_paths (Π W) · (A patterns)

 Π W     —— 语义空间上的累积变换（ℝᵈ，复合 ✓）
 A       —— 路由决策（ℝⁿ，不复合），各自取当前层的值
 Σ_paths —— 加法聚合（残差本身），连接不同路径
```

三种操作，三个代数角色，一个张量积空间。

---

## 边缘混沌：深度有效的动力学条件

残差流作为一个离散动力系统：

```
x^(l+1) = x^(l) + F(x^(l); W_Q, W_K, W_V, W_O, W_MLP)

Jacobian: J^(l) = I + ∂F/∂x|_{x=x^(l)}
Lyapunov 指数: λ_i = lim_{L→∞} (1/L) Σ_{l=1}^L ln σ_i(J^(l))
```

| 动力学状态 | Lyapunov 谱 | 对 Transformer 意味着什么 |
|---|---|---|
| λ_max ≪ 0 | 过阻尼 | 轨线收敛到不动点 → 深度坍缩 → 所有 token 最终同质 |
| λ_max ≈ 0 | 边缘混沌 | 轨线既不收敛也不发散 → 深度有持续信息增量 → 语义结构可维持 |
| λ_max ≫ 0 | 混沌 | 轨线指数发散 → 表示爆炸 → 初值极端敏感 → 不可训练 |

**训练 = 在参数空间中寻找边缘混沌态。** 训练好的 Transformer 必须恰好坐在 λ_max ≈ 0 的临界线上。

这解释了已知但未解释的现象：
- **Over-depth 退化**：层数超过 Lyapunov 时间后，轨线在混沌吸引子上随机游走，新层不增加信息
- **残差不能去掉**：去掉恒等项 I → Jacobian 失去对角的 I → 谱失控 → 无法维持在边缘混沌
- **LayerNorm 的位置敏感性**：LayerNorm = 每步对状态做投影 → 约束轨线不跑出有界区域 → 混沌保持"有界非周期"
- **越深越难训**：长轨线对参数极度敏感 → 反向传播通过边缘混沌系统 → 梯度本身也边缘混沌

## 不复合子空间的功能意义

```
如果 A 矩阵可复合（CK 方程成立）：
  A_eff = A^(L) · ... · A^(1)
  → Markov 链收敛到平稳分布
  → 所有 token 的注意力行相同
  → 所有 token 被同方向更新
  → 表示同质化
  → 语义聚类不存在
  → 深度没有意义（等价于直接采样平稳分布）

因为 A 矩阵不复合：
  A^(2) 作用于被第1层改变后的表示
  → 每层在处理"不同的东西"
  → 相似者趋同，相异者趋分（正反馈）
  → 语义聚类随深度增强
  → 深度有信息增量
```

**注意力态射复合的失败 = 语义聚类可以存在的数学前提。复合闭合 = 语义死亡。**
