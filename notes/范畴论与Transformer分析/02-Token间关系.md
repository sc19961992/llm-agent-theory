# Token 间关系——四种数学描述与聚类动力学

Token 间的关系从不同抽象层次看是四种不同的东西。每一种都不构成态射，但共同揭示一件事：**注意力在 ℝⁿ 上的路由 = 驱动 token 在 ℝᵈ 语义空间中移动的力。**

---

## 1. 作为矩阵分量：基底系数

```
a_{ij} = e_iᵀ A e_j = ⟨e_i, A e_j⟩
```

`a_{ij}` 是线性映射 A 在标准基下的矩阵表示的分量，不是独立的数学对象。换一组基底，`a_{ij}` 的值全变，但态射 A 不变。

**本质**：`a_{ij}` 是"选取观测坐标系后看到的一个投影值"，不是 token j 对 token i 的"作用"。

---

## 2. 作为图论结构：加权有向完全图

n 个 token 构成加权有向完全图：

```
顶点 V = {1, ..., n}
边 E = V × V
边权重 w(i→j) = a_{ij}
```

关键性质：
- **完全图**：每个 token 到每个其他 token 都有边
- **无组合律**：`w(i→j→k)` = w(i→j)·w(j→k) 作为矩阵乘积可计算，但乘积值不具有单层路由复合的语义——注意力仅在相邻 token 间定义，不存在"两步注意力"的概念
- **不是度量**：`a_{ij}` 不是距离，不满足对称性、非负性、三角不等式

图中可以讨论路径、流、割、谱性质，但这些是图的整体性质，不赋予单条边以态射身份。

---

## 3. 作为概率分布：条件权重

对固定的 i，`{a_{i1}, ..., a_{in}}` 经 softmax 后是概率分布：

```
P(token j 被关注 | token i 是查询) = softmax(Q_i · K_jᵀ / √d)_j
```

可做的事：
- **信息几何**：每个 token i 对应概率单纯形上的一个分布 `p_i`，分布间可通过 KL 散度、Fisher 信息度量、optimal transport 描述
- **检验非传递性**：`P(k|i) ≠ Σⱼ P(k|j) · P(j|i)`——这个不等式破坏的量本身就是有价值的信息

---

## 4. 作为交互能量的玻尔兹曼权重

软最大化前的原始分数 `s_{ij} = Q_i · K_jᵀ / √d` 定义能量函数：

```
E(i, j) = -s_{ij}
a_{ij} = exp(-E(i,j)) / Σ_k exp(-E(i,k))
```

- 高能量（低 `s_{ij}`）= token j 对 token i 几乎不可见
- 低能量（高 `s_{ij}`）= token j 的 V 向量大量累积到输出
- 温度 `√d` 控制分布锐度

---

## 综合对照

| 描述层次 | 数学对象 | Token i→j 是什么 | 复合律？ | 核心操作 |
|---------|---------|-----------------|---------|---------|
| 线性代数 | 矩阵分量 | 基底系数 | 矩阵乘法（A层面，非分量层面） | Σ_j a_{ij} v_j |
| 图论 | 邻接矩阵 | 有向边的权重 | 图乘积→多步路径（图上，非注意力上） | 谱分析、随机游走 |
| 信息几何 | 条件概率 | 分布 p(·&#124;i) 的第 j 分量 | 贝叶斯链（一般不满足） | KL 散度、OT 距离 |
| 统计力学 | 玻尔兹曼权重 | exp(-E(i,j))/Z | 无 | 温度缩放、能量最小化 |

**结论**：四种描述中，token 间关系的数学角色都是**某个更大结构的分量/元素/采样值**，不是独立的可复合态射。

---

## 但所有这些描述漏掉的东西：聚类动力学

上面的四种描述都只看了**一层**。跨层看，a_{ij} 不是固定值——它随表示演化：

```
a_{ij}^(l) 依赖 x_i^(l) 和 x_j^(l)
而 x_i^(l) 本身就依赖之前所有层的注意力模式
```

这产生了一个正反馈环：

```
token i 和 j 的表示相似
  → Q_i ≈ Q_j，K_i ≈ K_j
  → a_{i*} 的行 ≈ a_{j*} 的行（注意力模式相似）
  → Σ_k a_{ik} v_k ≈ Σ_k a_{jk} v_k（从上下文收到的信息相似）
  → x_i^(l+1) ≈ x_j^(l+1)（更新后仍然相似）
  → 聚类自持 ✓
```

反过来：

```
token i 和 j 的表示不同
  → 注意力模式不同
  → 被不同地更新
  → 表示更加不同
  → 聚类分化 ✓
```

**相似者趋同，相异者趋分。** 这是正反馈。注意力权重不是 token 间的固定关系——它们是**驱动 token 在语义空间 ℝᵈ 中移动的力的即时快照**，而这些力本身又依赖 token 当前在 ℝᵈ 中的位置。

注意力的非传递性（`a_{ik} ≠ something(a_{ij}, a_{jk})`）从这个角度看不是缺陷——**非传递性 = 正反馈不被"平均化"约束 = 聚类可以发生的条件。**
