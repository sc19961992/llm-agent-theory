# 十七条使用原则的统一证明

全部十七条原则仅依赖五个基础事实（均在论文§2中建立）：

1. 注意力输出概率分布（§2.1）：softmax每行自动归一化为概率，模型操作本质为P(next|context)
2. 训练收敛=梯度锻造，上下文不经此过程（§2.4）：参数经过了梯度反馈循环，上下文从未经历
3. 参数有效维度远小于名义维度→截断投影（§2.5）：d_eff << d_nominal，收敛覆盖有不可消除的边界
4. 记忆=空间结构vs顺序结构，无中间过渡（§2.6）：参数记忆和上下文记忆属于不同数学范畴
5. 多层残差流≠严格的映射复合（§2.7，诚实标注）：加性累积而非乘性复合，接受弱闭合

## 证明

### P1 精确优先于简短
错误累积∏(1-ε_k)随n单调递减+条件坐标以|V|^n膨胀，有效维度仅数百→n越小越好，每个保留的token须承载最大信息精度。

### P2 词汇的分布场效应
同音错别字证明模型不按词典定义选字，按P(字|上文)采样→两个字的条件分布越相似越可互换，与词典定义无关。词的"身份"=所有其他词到它的条件概率的总和。

### P3 目标约束而非路径约束
每增加m个路径指令→累积偏离风险从∏^n放大为∏^{n+m}。路径指令中每个词汇自带概率邻域（P2），指令越多模糊邻域引入越多。路径指令增加n和歧义密度；目标指令仅锚定输出端，不增加条件链偏离节点。

### P4 上下文信息的审慎选择
上下文信息未经梯度筛选（§2.4），注意力对所有token平等可及（§2.1），冗余与关键信息在softmax分母中零和竞争注意力份额。

### P5 上下文的分级滚动管理
P4在动态场景中的推广。固定容量+不加区分注意力→信息论最优策略为优先级分级滚动。

### P6 能力边界的节点划分
I(T)>E时部分约束不在窗口表示范围内→分解恢复每个约束的注意力可及性。P1约束n上界，P6约束信息量上界。

### P7 测试收敛驱动
参数冻结=训练收敛通道关闭→唯一残留修正机制为生成→验证→再生成循环。无验证信号时循环退化为无导向采样。

### P8 外部约束锚定
分解决策在上下文中做出，上下文未收敛→不可靠。父节点偏差δ沿层级乘性放大。外部锚定=唯一可切断此偏差传导链的信息源。

### P9 生成与检验的分离
检验本身是上下文推理（未收敛）。生成和检验共享上下文时，错误在相同条件分布下重现→自写自验=循环盲区。分离=检验信息集独立于生成过程的条件依赖。

### P10 逐环节验证
P8的时序操作化。偏差累积至∑δ_k∏c_i。逐环节验证在每层后截断偏差传播。

### P11 发散与收敛的空间隔离
偏好信号（相对序）和验证信号（绝对坐标）信息结构不同。共享上下文=用同一注意力对两种异质信息加权，相互稀释。

### P12 偏好驱动的对话式协作
C_V=∅→测试收敛失去工作条件。偏好信号是唯一残留的导向源。操作者在线性=收敛循环闭合条件的直接推论。

### P13 扁平化管理
多层嵌套=在未收敛的上下文中做高层决策+高层窗口容量被下层元数据耗竭。单层扁平=k被约束为1。

### P14 操作者上限
Agent的每个不可自举节点均由操作者填补→操作者在这些节点处的认知质量=系统在这些维度上的能力上界。

### P15 验证覆盖即能力边界
测试收敛有效性条件为C_V=C。|C_V|/|C|是Agent可靠性的结构上界——信息论恒等式。

### P16 根需求的外在性
验证链顶端判断"用户真正要什么"的语义不在训练分布中。根需求语义从未作为训练目标→验证链在此不可闭合。

### P17 幻觉边界不可消除
softmax必输出+d_eff始终有限→覆盖始终有边界→边界外幻觉不可消除。可缩小范围，不可消除边界本身。
