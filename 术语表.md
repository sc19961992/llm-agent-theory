## 核心概念

### 注意力机制
- **定义**：替代RNN循环理念，触及思维结构中自然语言部分的核心架构。单层softmax注意力严格产出随机矩阵（Markov核）。
- **关系**：→ 贝叶斯推断 → 态射 → 截断投影

### 贝叶斯推断（注意力等价）
- **定义**：注意力机制下的Transformer实质在做贝叶斯推断——以训练分布统计为隐式先验的核条件期望。
- **三层结构**：
  1. 单头单层注意力 → 单次贝叶斯推断快照
  2. 多层Transformer前馈 → 多轮信念精炼（类变分推断迭代）
  3. 全程预训练 → 隐式贝叶斯模型平均
- **弱闭合**：第二层残差加法≠态射乘法（Chapman-Kolmogorov），仅功能逼近。

### 米田嵌入
- **定义**：一个token的语义身份由所有其他token到它的条件概率关系所穷尽定义。
- **等价表述**：分布假说（Firth 1957）的精确概率形式。
- **关系**：语义空间 = 米田 + 贝叶斯关系总和

### 态射（抽象态射）
- **定义**：提示词到语义空间的映射。单层注意力=FinStoch态射（数学事实）。
- **注意**：多层不等于态射可乘复合（残差加法累积 ≠ 矩阵乘法链）。

### 函子
- **定义**：训练收敛找到的(数据,关系)→(预测,关系)近似结构保持映射。
- **弱闭合**：交叉熵不蕴含函子性（F(g∘f)≠F(g)∘F(f)），经验发现而非定义保证。

### 截断投影
- **定义**：参数空间有效维度有限→永远只能学到米田嵌入的低维截断投影，而非完整函子。
- **等价表述**：低秩近似（low-rank approximation of semantic mapping）
- **核心约束**：超出截断投影范围的语义态射不受收敛保证。
- **数学硬伤**：FinStoch本身是有限维的→"截断"作为原则性限制需重建论证（拆分两层语义空间）。

### 收敛（训练收敛）
- **定义**：基于梯度迭代找到结构保持映射的优化过程。
- **三条判据**：损失稳定、参数更新趋零、行为可预测
- **关键属性**：参数一次收敛后静态冻结，不再随使用演化。

### 测试收敛
- **定义**：参数冻结后，Agent通过"生成→验证→修正→再验证"循环逼近目标的替代收敛机制。
- **关键对比**：训练收敛=梯度下降（有单调保证）；测试收敛=验证反馈迭代（无梯度、无单调保证）。
- **等价表述**：verification-guided iterative refinement

### 收敛约束 / 收敛覆盖
- **定义**：经历梯度锻造后享有的结构稳定性及其覆盖范围。
- **等价表述**：learned mapping reliability domain / in-distribution generalization region
- **边界性质**：边界永远存在，不可消除，只可推移。

### 记忆二分（长期/短期）
- **长期记忆**：基座模型+强化学习共同形成的参数空间表示。收敛一次→参数冻结。静态、空间结构。
- **短期记忆**：上下文环境中的序列化信息。动态、顺序结构、长度有限。
- **核心断言**：二者属不同数学结构→不存在连续过渡的"中期记忆"。
- **时间性缺失**：位置编码提供顺序，不提供时间深度（无衰减、无巩固、无自发重组）。

### Agent = 模型 + Harness
- **Harness**：程序框架 + 上下文环境
- **比喻**：模型=远程操控者，Harness=机甲

### 收敛任务
- **定义**：目标可被可自动执行验证标准完整刻画的Agent任务。
- **等价表述**：goal-directed execution
- **特征**：前置目标、可验证、有潜在路径。

### 发散任务
- **定义**：无可自动执行验证标准的任务，目标在探索中成形，依赖偏好信号驱动。
- **等价表述**：open-ended exploration
- **特征**：无前置目标、无自动验证、偏好信号=唯一方向梯度。

### 根需求
- **定义**：验证链递归上溯的终点——"用户真正需要什么"。语义在训练分布之外。
- **等价表述**：terminal verification criteria（AI对齐文献: value specification problem / scalable oversight）
- **命题**：根需求外在定律——最高层目标必须由外部（人）提供。

### 关键分叉点
- **定义**：不可逆的任务结构决策节点（需求消歧、接口定义、模块边界）。
- **特征**：未经收敛的上下文驱动→不可靠是结构必然。
- **原则**：须由人工或程序硬约束锚定，不由模型自行判断。

### 级联传导
- **定义**：父节点错误一次性向下传播，子节点全偏且无独立纠正信号。
- **量化**：修正复杂度指数级上升。

### 偏好信号
- **定义**：发散任务中人对候选方向的比较性判断（"A vs B"）。
- **等价**：采样器的方向梯度，类比收敛任务中损失函数的梯度。
- **约束**：稀疏→停滞；矛盾→震荡；分布外→退化。

### 发散收敛分池
- **定义**：发散探索和收敛执行使用各自独立的上下文空间，仅通过可验证目标结晶后显式过渡。
- **理由**：偏好标注与验证状态相互污染。

### 写测分离
- **定义**：生成输出与检验输出不由同一模型实例在同一上下文中完成。
- **理由**：检验过程同样在上下文中，同样未经收敛→自写自验是循环盲区。

### 人定上限规律
- **定义**：操作人的理解力和判断力直接划定Agent在该任务上的实际能力边界。
- **理由**：根需求外在 + 节点不可靠 + 偏好唯一 → 人的上限即系统上限。

### 验证即边界规则
- **定义**：Agent能可靠做什么，不超过你能对该任务自动检验什么。
- **理由**：测试收敛的有效性=验证信号的覆盖度。

### 扁平化自动管理原则
- **定义**：子Agent管理采用单层级扁平结构，避免多层嵌套。
- **理由**：每下一层级自动提示词不可靠性倍增，高层收敛成本累积超上下文窗口承载极限。

### AGI不可达
- **定义**：LLM Agent在当前Transformer训练范式下不可能实现人类意义上的AGI。
- **五项缺失**：语义空间不完备（截断投影）、模态不对齐（注意力限于语言）、动机不自发（根因未明）、无渐进记忆（记忆二分）、顺序性而非时间性（位置编码）。

### 幻觉不可消灭只可推移
- **定义**：收敛覆盖范围外必然产生不可靠输出。边界可缩小，不可消除。
- **理由**：收敛覆盖是截断投影→边界永远存在→边界外无收敛约束。

### 词态射即场合
- **定义**：词语靠出没的语境起作用，不靠字面意思。词的态射指向它常出现的语用场子，不指向它的语义定义。
- **原理**：米田引理操作化——一个对象由所有指向它的关系所定义。
